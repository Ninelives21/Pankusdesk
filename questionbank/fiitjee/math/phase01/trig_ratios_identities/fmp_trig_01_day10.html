\(\implies \cos A \cos^2 B \cos C + \sin A \cos^2 B \sin C - \cos A \sin^2 B \cos C - \sin A \sin^2 B \sin C = \cos A \cos C - \sin A \sin C \)

\(\implies \cos A \cos^2 B \cos C + \sin A \cos^2 B \sin C - \cos A \sin^2 B \cos C - \sin A \sin^2 B \sin C - \cos A \cos C + \sin A \sin C = 0 \)
