To find the locus of a point that moves equidistant from the lines \(3x + 4y + 5 = 0\) and \(2x - 3y + 6 = 0\), we use the formula for the perpendicular distance from a point \((x_1, y_1)\) to a line \(ax + by + c = 0\): \[ d = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2 + b^2}} \] Let \((x, y)\) be the point that is equidistant from both lines. Then the perpendicular distance from \((x, y)\) to \(3x + 4y + 5 = 0\) is equal to the perpendicular distance from \((x, y)\) to \(2x - 3y + 6 = 0\): \[ \frac{|3x + 4y + 5|}{\sqrt{3^2 + 4^2}} = \frac{|2x - 3y + 6|}{\sqrt{2^2 + 3^2}} \] Simplify the equation: \[ \frac{|3x + 4y + 5|}{\sqrt{9 + 16}} = \frac{|2x - 3y + 6|}{\sqrt{4 + 9}} \] \[ \frac{|3x + 4y + 5|}{5} = \frac{|2x - 3y + 6|}{\sqrt{13}} \] Cross-multiply to eliminate the denominators: \[ \sqrt{13} \cdot |3x + 4y + 5| = 5 \cdot |2x - 3y + 6| \] Squaring both sides to remove the absolute values: \[ 13 (3x + 4y + 5)^2 = 25 (2x - 3y + 6)^2 \] Expanding both sides: \[ 13 (9x^2 + 24xy + 16y^2 + 30x + 40y + 25) = 25 (4x^2 - 12xy + 9y^2 + 24x - 36y + 36) \] Simplifying: \[ 117x^2 + 312xy + 208y^2 + 390x + 520y + 325 = 100x^2 - 300xy + 225y^2 + 600x - 900y + 900 \] Rearrange all terms to one side: \[ 117x^2 + 312xy + 208y^2 + 390x + 520y + 325 - 100x^2 - 300xy - 225y^2 - 600x + 900y - 900 = 0 \] Combine like terms: \[ 17x^2 + 612xy - 17y^2 - 210x + 1420y - 575 = 0 \]

To find the locus of a point that moves equidistant from the line \(2x + 3y + 6 = 0\) at a distance of 5, we use the distance formula. The perpendicular distance \(d\) of a point \((x, y)\) from a line \(ax + by + c = 0\) is given by: \[ d = \frac{|2x + 3y + 6|}{\sqrt{2^2 + 3^2}} = 5 \] Simplifying, \[ 5 = \frac{|2x + 3y + 6|}{\sqrt{13}} \] Multiplying both sides by \(\sqrt{13}\): \[ 5\sqrt{13} = |2x + 3y + 6| \] This absolute value equation gives us two cases: Case 1: \[ 2x + 3y + 6 = 5\sqrt{13} \] Case 2: \[ 2x + 3y + 6 = -5\sqrt{13} \] Squaring both sides to remove the absolute values and simplifying: \[ (2x + 3y + 6)^2 = (5\sqrt{13})^2 \] \[ 4x^2 + 9y^2 + 36 + 12xy + 36y + 24x = 25 \times 13 \] Simplifying further: \[ 4x^2 + 9y^2 + 12xy + 24x + 36y - 289 = 0 \] Therefore, the locus of the point is given by: \[ 4x^2 + 9y^2 + 12xy + 24x + 36y - 289 = 0 \]

Given: \[ x = a \sin \theta \quad \text{and} \quad y = b \cos \theta \] Squaring both equations: \[ x^2 = a^2 \sin^2 \theta \quad \text{and} \quad y^2 = b^2 \cos^2 \theta \] Dividing both sides by \(a^2\) and \(b^2\) respectively: \[ \sin^2 \theta = \frac{x^2}{a^2} \quad \text{and} \quad \cos^2 \theta = \frac{y^2}{b^2} \] Adding the equations: \[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = \sin^2 \theta + \cos^2 \theta \] Since \(\sin^2 \theta + \cos^2 \theta = 1\), we get: \[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \] Therefore, the locus of the point is: \[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \]
