Concept notes

Structural Isomerism

iso-structural

Structural isomers have the same molecular formula, but the atoms are connected in a different order. For exam questions, first check whether the carbon skeleton, functional group, or position has changed.

Definition Same molecular formula, different connectivity between atoms.
Common trap Do not call every different-looking structure a new isomer. First check if it is just the same structure drawn differently.
Lecture link: Alakh Pandey Isomerism Lecture • timestamp to be added

Geometrical Isomerism

iso-geometrical

Geometrical isomerism appears when rotation is restricted and each double-bonded carbon has suitable different groups. The main mistake is checking only the double bond, but not checking both sides properly.

Quick check Restricted rotation + suitable different groups on both ends.
