    • Correct answer: (b) \(-1.962 \times 10^{-17}\ \mathrm{J}\)
    • Explanation: \(\mathrm{Li^{2+}}\) is a hydrogen-like species because it has only one electron. For any hydrogen-like species, the energy of the \(n\)th orbit is given by \(E_n = -2.18 \times 10^{-18}\left(\dfrac{Z^2}{n^2}\right)\ \mathrm{J}\). For \(\mathrm{Li^{2+}}\), \(Z=3\) and for first orbit \(n=1\), so the energy becomes \(-2.18 \times 10^{-18} \times 9 = -1.962 \times 10^{-17}\ \mathrm{J}\).
    • Step-by-step explanation
      • This is a direct Bohr model question on hydrogen-like ions.
      • \(\mathrm{Li^{2+}}\) has atomic number \(Z=3\).
      • Neutral Li has 3 electrons, but \(\mathrm{Li^{2+}}\) has lost 2 electrons, so it has only 1 electron left.
      • That means it behaves like hydrogen, except the nucleus has charge \(+3\).
      • For a hydrogen-like species:
        • \(E_n = -2.18 \times 10^{-18}\left(\dfrac{Z^2}{n^2}\right)\ \mathrm{J}\)
      • Here:
        • \(Z = 3\)
        • \(n = 1\)
      • So,
        • \(E_1 = -2.18 \times 10^{-18}\left(\dfrac{3^2}{1^2}\right)\)
        • \(E_1 = -2.18 \times 10^{-18} \times 9\)
        • \(E_1 = -19.62 \times 10^{-18}\ \mathrm{J}\)
        • \(E_1 = -1.962 \times 10^{-17}\ \mathrm{J}\)
    • Concept used / theory behind it
      • In the Bohr model, energy depends on:
        • nuclear charge \(Z\)
        • orbit number \(n\)
      • The energy becomes more negative when:
        • \(Z\) increases
        • \(n\) decreases
      • So the first orbit of \(\mathrm{Li^{2+}}\) must be much lower in energy than the second orbit of hydrogen.
      • The negative sign means the electron is bound to the nucleus.
    • Why the correct option is right
      • Using the standard formula directly gives:
        • \(-1.962 \times 10^{-17}\ \mathrm{J}\)
      • This matches option (b).
    • Why other options are wrong
      • (a) is smaller by a factor of 10.
      • (c) is double the correct value.
      • (d) is again wrong in magnitude.
      • These are typical calculation-slip options.
    • Exam tip / shortcut / elimination clue
      • For hydrogen-like ions, always remember:
        • \(E_n \propto -\dfrac{Z^2}{n^2}\)
      • If \(Z=3\) and \(n=1\), the factor is \(9\).
      • So just multiply \(-2.18 \times 10^{-18}\) by 9 quickly.
    • Final result
      • Energy of first orbit of \(\mathrm{Li^{2+}}\) \(= -1.962 \times 10^{-17}\ \mathrm{J}\).

    • Explanation: Atomic number 24 is chromium. Its electronic configuration is \(\mathrm{1s^2\,2s^2\,2p^6\,3s^2\,3p^6\,4s^1\,3d^5}\). Now we group electrons according to the value of \((n+l)\). For \((n+l)=3\), the subshells are \(2p\) and \(3s\), giving \(6+2=8\) electrons. For \((n+l)=4\), the subshells are \(3p\) and \(4s\), giving \(6+1=7\) electrons. For \((n+l)=5\), the relevant occupied subshell is \(3d\), giving 5 electrons. Hence the answer is \(8,7,5\).
    • Step-by-step explanation
      • First identify the element with \(Z=24\).
      • \(Z=24\) means chromium, \(\mathrm{Cr}\).
      • Its actual electronic configuration is:
        • \(\mathrm{Cr = 1s^2\,2s^2\,2p^6\,3s^2\,3p^6\,4s^1\,3d^5}\)
      • This is an important exceptional configuration.
      • Now calculate \((n+l)\) for occupied subshells
        • For \(s\)-subshell, \(l=0\)
        • For \(p\)-subshell, \(l=1\)
        • For \(d\)-subshell, \(l=2\)
      • For \((n+l)=3\)
        • \(2p:\ n=2,\ l=1 \Rightarrow n+l=3\)
        • \(3s:\ n=3,\ l=0 \Rightarrow n+l=3\)
        • Electrons present:
          • \(2p^6 \Rightarrow 6\)
          • \(3s^2 \Rightarrow 2\)
        • Total \(= 6+2 = 8\)
      • For \((n+l)=4\)
        • \(3p:\ n=3,\ l=1 \Rightarrow 4\)
        • \(4s:\ n=4,\ l=0 \Rightarrow 4\)
        • Electrons present:
          • \(3p^6 \Rightarrow 6\)
          • \(4s^1 \Rightarrow 1\)
        • Total \(= 6+1 = 7\)
      • For \((n+l)=5\)
        • \(3d:\ n=3,\ l=2 \Rightarrow 5\)
        • Electrons present:
          • \(3d^5 \Rightarrow 5\)
        • Total \(= 5\)
    • Concept used / theory behind it
      • The \((n+l)\) rule is usually used to predict the order of filling of orbitals.
      • Here the question uses it differently: it asks how many electrons belong to subshells having a certain \((n+l)\) value.
      • The key is:
        • write correct electronic configuration first
        • then identify occupied subshells matching the required \((n+l)\)
      • Chromium is a trap because many students write \(\mathrm{4s^2\,3d^4}\), but the correct configuration is \(\mathrm{4s^1\,3d^5}\).
    • Why the correct option is right
      • We found:
        • \((n+l)=3 \Rightarrow 8\)
        • \((n+l)=4 \Rightarrow 7\)
        • \((n+l)=5 \Rightarrow 5\)
      • So the correct sequence is \(8,7,5\), which matches option (c).
    • Why other options are wrong
      • Options with 8 for \((n+l)=4\) usually come from wrongly writing chromium as \(\mathrm{4s^2\,3d^4}\).
      • Any option giving 6 for \((n+l)=5\) is also incorrect because chromium has \(3d^5\), not \(3d^6\).
    • Exam tip / shortcut / elimination clue
      • Always remember the common exceptions:
        • \(\mathrm{Cr = 4s^1\,3d^5}\)
        • \(\mathrm{Cu = 4s^1\,3d^{10}}\)
      • That one fact is enough to save this question.
    • Final result
      • The required numbers are \(8, 7, 5\).

Therefore the matching is A-IV, B-III, C-I, D-II.
    • Step-by-step explanation
      • Let us decode each atomic number one by one.
      • A. Atomic number 56
        • \(56\) is barium, \(\mathrm{Ba}\).
        • Electronic configuration:
          • \(\mathrm{Ba = (Xe)\,6s^2}\)
        • It belongs to the \(s\)-block.
        • Since outermost configuration is \(6s^2\):
          • group \(=2\)
          • period \(=6\)
        • So A \(\rightarrow\) IV \((2,6)\)
      • B. Atomic number 50
        • \(50\) is tin, \(\mathrm{Sn}\).
        • Electronic configuration:
          • \(\mathrm{Sn = (Kr)\,4d^{10}5s^2 5p^2}\)
        • It is a \(p\)-block element.
        • For \(p\)-block, group number is \(10 +\) total valence electrons in \(ns\) and \(np\).
        • Here valence part is \(5s^2 5p^2\), so:
          • group \(= 10 + 4 = 14\)
          • period \(= 5\)
        • So B \(\rightarrow\) III \((14,5)\)
      • C. Atomic number 27
        • \(27\) is cobalt, \(\mathrm{Co}\).
        • Electronic configuration:
          • \(\mathrm{Co = (Ar)\,3d^7 4s^2}\)
        • It is a \(d\)-block element.
        • For transition elements, group number \(=\) \((n-1)d\) electrons \(+\ ns\) electrons.
        • So:
          • group \(= 7+2 = 9\)
          • period \(= 4\)
        • So C \(\rightarrow\) I \((9,4)\)
      • D. Atomic number 58
        • \(58\) is cerium, \(\mathrm{Ce}\).
        • Electronic configuration is written approximately as:
          • \(\mathrm{Ce = (Xe)\,4f^1 5d^1 6s^2}\)
        • It is an \(f\)-block element of the lanthanoid series.
        • In this classification:
          • group \(= 3\)
          • period \(= 6\)
        • So D \(\rightarrow\) II \((3,6)\)
    • Concept used / theory behind it
      • This question tests periodic classification using electronic configuration.
      • Quick rules:
        • \(s\)-block: group from \(ns^1\) or \(ns^2\)
        • \(p\)-block: group \(= 10 + (ns + np)\) electrons
        • \(d\)-block: group \(= (n-1)d + ns\) electrons
        • \(f\)-block lanthanoids are commonly placed in group 3, period 6
      • Period number is the highest principal quantum number present in the outer shell.
    • Why the correct option is right
      • We obtained:
        • A \(\rightarrow\) IV
        • B \(\rightarrow\) III
        • C \(\rightarrow\) I
        • D \(\rightarrow\) II
      • This exactly matches option (d).
    • Why other options are wrong

        • Ba as group 2 period 6
        • Sn as group 14 period 5
        • Co as group 9 period 4
        • Ce as group 3 period 6

    • Exam tip / shortcut / elimination clue
      • Memorise a few anchor elements:
        • Ba \(\rightarrow\) group 2, period 6
        • Sn \(\rightarrow\) group 14, period 5
        • Co \(\rightarrow\) group 9, period 4
        • Ce \(\rightarrow\) lanthanoid, group 3, period 6
      • That alone makes this match-the-following question very quick.

      • The correct matching is A-IV, B-III, C-I, D-II.

    • Correct answer: (a) \(\mathrm{C_2}\)
    • Explanation: According to molecular orbital theory, \(\mathrm{C_2}\) has the configuration \(\sigma1s^2\,\sigma^{*}1s^2\,\sigma2s^2\,\sigma^{*}2s^2\,\pi2p_x^2\,\pi2p_y^2\). The bonding electrons responsible for bond formation are only in the two \(\pi\) orbitals, while the \(\sigma2p\) orbital is empty. So the bond between the two carbon atoms consists only of \(\pi\)-bonds.
    • Step-by-step explanation
      • We compare the molecular orbital filling of the given diatomic molecules.
      • For lighter molecules up to nitrogen, the order of \(2p\) molecular orbitals is:
        • \(\pi2p_x = \pi2p_y < \sigma2p_z\)
      • For \(\mathrm{C_2}\)
        • Total electrons \(= 12\).
        • Molecular orbital configuration:
          • \(\sigma1s^2\,\sigma^{*}1s^2\,\sigma2s^2\,\sigma^{*}2s^2\,\pi2p_x^2\,\pi2p_y^2\)
        • The \(\sigma2p_z\) orbital is still empty.
        • So the bond arises only from two filled \(\pi\)-bonding orbitals.
        • That means, in MOT language, \(\mathrm{C_2}\) contains only \(\pi\)-bonds between the atoms.
      • Why not \(\mathrm{N_2}\)?
        • \(\mathrm{N_2}\) has one additional filled \(\sigma2p_z\) orbital.
        • So it contains:
          • one \(\sigma\)-bond
          • two \(\pi\)-bonds
        • Therefore it does not contain only \(\pi\)-bonds.
      • Why not \(\mathrm{O_2}\)?
        • \(\mathrm{O_2}\) also has a \(\sigma2p_z\) bond present.
        • So it is not a “only \(\pi\)-bonds” case.
      • Why not \(\mathrm{B_2}\)?
        • \(\mathrm{B_2}\) has fewer electrons and only one electron each in the two \(\pi\) orbitals.
        • It does not correspond to the same fully paired two-\(\pi\)-bond situation asked here.
    • Concept used / theory behind it
      • Molecular orbital theory describes bonding in terms of electrons occupying molecular orbitals formed by overlap of atomic orbitals.
      • A \(\sigma\)-bond comes from head-on overlap, while a \(\pi\)-bond comes from sidewise overlap.
      • For \(\mathrm{C_2}\), after cancellation of \(1s\) and \(2s\) bonding-antibonding pairs, the remaining bonding electrons occupy only \(\pi2p\) orbitals.
      • That is why \(\mathrm{C_2}\) is a special and often-tested case.
    • Why the correct option is right
      • \(\mathrm{C_2}\) has bonding due only to \(\pi2p_x\) and \(\pi2p_y\) orbitals.
      • No \(\sigma2p\) bonding orbital is occupied.
      • Hence it is the correct answer.
    • Why other options are wrong
      • \(\mathrm{N_2}\) has a \(\sigma\)-bond also.
      • \(\mathrm{O_2}\) has \(\sigma\)-bonding as well as \(\pi\)-bonding.
      • \(\mathrm{B_2}\) does not fit the asked condition in the usual exam treatment.
    • Exam tip / shortcut / elimination clue
      • Memorise this special MOT fact:
        • \(\mathrm{C_2}\) is the classic molecule said to contain only \(\pi\)-bonds in entrance-exam questions.
      • If this exact statement appears, \(\mathrm{C_2}\) is usually the intended answer.
    • Final result
      • According to molecular orbital theory, the molecule containing only \(\pi\)-bonds is \(\mathrm{C_2}\).

    • Correct answer: (d) \(\mathrm{NH_3 + H^+ \rightarrow NH_4^+}\)
    • Explanation: In \(\mathrm{NH_3}\), nitrogen is \(sp^3\)-hybridised, and in \(\mathrm{NH_4^+}\) also nitrogen remains \(sp^3\)-hybridised. Only the number of lone pairs and bond pairs changes, but the hybridisation does not change. Hence option (d) is correct.
    • Geometry and hybridisation of \(\mathrm{NH_3}\) and \(\mathrm{NH_4^+}\)
    • Step-by-step explanation
      • This question is about comparing the hybridisation of the central atom before and after change.
      • Let us check each option carefully.
      • (a) \(\mathrm{C_2H_4 \rightarrow C_2H_6}\)
        • In ethene \(\mathrm{(C_2H_4)}\), each carbon is \(sp^2\)-hybridised.
        • In ethane \(\mathrm{(C_2H_6)}\), each carbon is \(sp^3\)-hybridised.
        • So hybridisation changes from \(sp^2\) to \(sp^3\).
      • (b) \(\mathrm{PCl_5 + Cl^- \rightarrow PCl_6^-}\)
        • In \(\mathrm{PCl_5}\), phosphorus is \(sp^3d\)-hybridised with trigonal bipyramidal geometry.
        • In \(\mathrm{PCl_6^-}\), phosphorus becomes \(sp^3d^2\)-hybridised with octahedral geometry.
        • So hybridisation changes.
      • (c) \(\mathrm{BF_3 + F^- \rightarrow BF_4^-}\)
        • In \(\mathrm{BF_3}\), boron is \(sp^2\)-hybridised and trigonal planar.
        • In \(\mathrm{BF_4^-}\), boron is \(sp^3\)-hybridised and tetrahedral.
        • So hybridisation changes from \(sp^2\) to \(sp^3\).
      • (d) \(\mathrm{NH_3 + H^+ \rightarrow NH_4^+}\)
        • In \(\mathrm{NH_3}\), nitrogen has 3 bond pairs and 1 lone pair.
        • The steric number is \(4\), so nitrogen is \(sp^3\)-hybridised.
        • In \(\mathrm{NH_4^+}\), nitrogen has 4 bond pairs and 0 lone pairs.
        • The steric number is still \(4\), so nitrogen remains \(sp^3\)-hybridised.
        • Thus hybridisation does not change.
    • Concept used / theory behind it
      • Hybridisation depends mainly on the steric number, that is:
        • number of \(\sigma\)-bonds
        • plus number of lone pairs on the central atom
      • If the total electron domains remain the same, hybridisation usually remains the same.
      • Shape may change, however, because shape depends on how many of those electron pairs are lone pairs.
      • This is exactly what happens in \(\mathrm{NH_3}\) and \(\mathrm{NH_4^+}\):
        • \(\mathrm{NH_3}\): \(sp^3\), trigonal pyramidal
        • \(\mathrm{NH_4^+}\): \(sp^3\), tetrahedral
    • Why the correct option is right
      • In both \(\mathrm{NH_3}\) and \(\mathrm{NH_4^+}\), nitrogen has steric number \(4\).
      • Therefore nitrogen stays \(sp^3\)-hybridised in both species.
      • So option (d) is the only case where there is no change in hybridisation.
    • Why other options are wrong
      • (a) changes from \(sp^2\) to \(sp^3\).
      • (b) changes from \(sp^3d\) to \(sp^3d^2\).
      • (c) changes from \(sp^2\) to \(sp^3\).
    • Exam tip / shortcut / elimination clue
      • Do not look only at the shape.
      • Always count total electron domains around the central atom.
      • If electron domains stay the same, hybridisation often stays the same even if the shape changes.
      • Classic example:
        • \(\mathrm{NH_3}\) and \(\mathrm{NH_4^+}\) are both \(sp^3\)
    • Final result
      • No change in hybridisation occurs in \(\mathrm{NH_3 + H^+ \rightarrow NH_4^+}\).

    • Explanation: By Graham’s law, rate of diffusion is inversely proportional to the square root of molar mass. So \(\dfrac{r_A}{r_B} = \sqrt{\dfrac{M_B}{M_A}}\). Given \(\dfrac{r_A}{r_B} = \sqrt{5}\) and \(M_A = x\), we get \(\sqrt{5} = \sqrt{\dfrac{M_B}{x}}\). Squaring both sides gives \(M_B = 5x\).
    • Step-by-step explanation
      • Use Graham’s law of diffusion:
        • \(r \propto \dfrac{1}{\sqrt{M}}\)
      • Therefore, for two gases:
        • \(\dfrac{r_A}{r_B} = \sqrt{\dfrac{M_B}{M_A}}\)
      • The question says gas \(A\) diffuses \(\sqrt{5}\) times as fast as gas \(B\).
      • So:
        • \(\dfrac{r_A}{r_B} = \sqrt{5}\)
      • Also given:
        • \(M_A = x\)
      • Substitute into Graham’s law:
        • \(\sqrt{5} = \sqrt{\dfrac{M_B}{x}}\)
      • Now square both sides:
        • \(5 = \dfrac{M_B}{x}\)
      • So:
        • \(M_B = 5x\)
    • Concept used / theory behind it
      • Graham’s law tells us:
        • lighter gases diffuse faster
        • heavier gases diffuse slower
      • The relation is not linear.
      • It depends on the square root of molar mass.
      • So whenever the rate ratio is given as a square root, squaring is usually the key step.
    • Why the correct option is right
      • Only option (b) satisfies:
        • \(\dfrac{r_A}{r_B} = \sqrt{\dfrac{M_B}{M_A}} = \sqrt{5}\)
      • That requires \(M_B = 5x\).
    • Why other options are wrong
      • (a) would give \(\sqrt{4} = 2\), not \(\sqrt{5}\).
      • (c) would give \(\sqrt{16} = 4\).
      • (d) would give \(\sqrt{25} = 5\).
      • So these do not match the given rate ratio.
    • Exam tip / shortcut / elimination clue
      • If the faster gas is \(A\), then \(A\) must be lighter.
      • So \(M_B\) must be greater than \(M_A\).
      • That immediately eliminates any option smaller than or equal to \(x\).
      • Then square the rate ratio:
        • \((\sqrt{5})^2 = 5\)
    • Final result
      • Molar mass of gas \(B = 5x\ \mathrm{g\ mol^{-1}}\).

    • Step-by-step explanation
      • A buffer resists change in pH when a small amount of acid or base is added.
      • To form a buffer, we need one of these two combinations:
        • weak acid + its salt with strong base
        • weak base + its salt with strong acid
      • Option (a): \(\mathrm{NH_3 + HCl}\) in \(2:1\) ratio
        • \(\mathrm{NH_3}\) is a weak base and \(\mathrm{HCl}\) is a strong acid.
        • One mole of \(\mathrm{HCl}\) neutralises one mole of \(\mathrm{NH_3}\), leaving one mole of \(\mathrm{NH_3}\) unreacted.
        • Final mixture contains:
          • \(\mathrm{NH_3}\) (weak base)
          • \(\mathrm{NH_4Cl}\) (its conjugate acid salt)
        • So this is a basic buffer.
      • Option (b): \(\mathrm{CH_3CO_2H + NaOH}\) in \(2:1\) ratio
        • \(\mathrm{CH_3COOH}\) is a weak acid and \(\mathrm{NaOH}\) is a strong base.
        • One mole of \(\mathrm{NaOH}\) neutralises one mole of acetic acid, leaving one mole of acetic acid unreacted.
        • Final mixture contains:
          • \(\mathrm{CH_3COOH}\) (weak acid)
          • \(\mathrm{CH_3COONa}\) (its conjugate base salt)
        • So this is an acidic buffer.
      • Option (c): \(\mathrm{NaOH + CH_3COOH}\) in \(1:1\) ratio
        • Here the weak acid and strong base are in exactly equivalent amounts.
        • So complete neutralisation occurs:
          • \(\mathrm{CH_3COOH + NaOH \rightarrow CH_3COONa + H_2O}\)
        • After the reaction, only sodium acetate remains; acetic acid is absent.
        • Since both weak acid and conjugate base are not present together, it is not a buffer.
      • Option (d): \(\mathrm{NH_4Cl + NH_3}\) in \(1:1\) ratio
        • This directly contains:
          • \(\mathrm{NH_3}\) (weak base)
          • \(\mathrm{NH_4Cl}\) (conjugate acid salt)
        • So it is a basic buffer.
    • Concept used / theory behind it
      • A buffer needs two partners that can neutralise added acid or base.
      • That is why we need:
        • \(\mathrm{HA}\) and \(\mathrm{A^-}\), or
        • \(\mathrm{B}\) and \(\mathrm{BH^+}\)
      • If the weak acid is fully consumed, only salt remains, and the solution loses buffer action.
      • Similarly, if the weak base is fully consumed, no basic buffer remains.
    • Why the correct option is right
      • In option (c), equal moles of \(\mathrm{NaOH}\) and \(\mathrm{CH_3COOH}\) react completely.
      • The final solution does not contain both \(\mathrm{CH_3COOH}\) and \(\mathrm{CH_3COO^-}\) together.
      • Hence it does not form a buffer.
    • Why other options are wrong
      • (a) gives \(\mathrm{NH_3/NH_4^+}\), so it is a buffer.
      • (b) gives \(\mathrm{CH_3COOH/CH_3COO^-}\), so it is a buffer.
      • (d) is directly a weak base and its conjugate acid salt, so it is a buffer.

        • unequal ratio with acid in excess \(\rightarrow\) buffer
        • exact \(1:1\) neutralisation \(\rightarrow\) not a buffer
      • So if weak acid and strong base are in equal moles, be very careful: that is often the non-buffer option.
    • Final result
      • The mixture that does not form a buffer solution is (c).
