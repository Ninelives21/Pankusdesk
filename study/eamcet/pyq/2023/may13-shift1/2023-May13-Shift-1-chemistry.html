    • Correct answer: (c) \((2),\ Be^{3+}\).
    • Explanation: For a hydrogen-like species, Bohr radius of the \(n\)th orbit is given by \(r_n = 0.529\left(\dfrac{n^2}{Z}\right)\ \mathring{A}\). For hydrogen in the first orbit, \(n=1\) and \(Z=1\), so \(r_H = 0.529\ \mathring{A}\). To get the same radius for another hydrogen-like ion, we need \(\dfrac{n^2}{Z}=1\). For \(Be^{3+}\), \(Z=4\), and with \(n=2\), \(\dfrac{2^2}{4}=1\). So the radius matches exactly.
    • Step-by-step explanation
      • Use the Bohr orbit radius formula for hydrogen-like species:
        • \(r_n = 0.529\left(\dfrac{n^2}{Z}\right)\ \mathring{A}\)
      • For hydrogen atom in the first orbit:
        • \(n=1,\ Z=1\)
        • \(r_H = 0.529\left(\dfrac{1^2}{1}\right)=0.529\ \mathring{A}\)
      • Now check the options one by one.
      • Option (a): \((2),\ Li^{2+}\)
        • Here \(Z=3\)
        • \(r = 0.529\left(\dfrac{2^2}{3}\right)=0.529\left(\dfrac{4}{3}\right)\)
        • This is not equal to \(0.529\)
      • Option (b): \((3),\ Li^{2+}\)
        • \(r = 0.529\left(\dfrac{3^2}{3}\right)=0.529(3)\)
        • Not equal to \(0.529\)
      • Option (c): \((2),\ Be^{3+}\)
        • Here \(Z=4\)
        • \(r = 0.529\left(\dfrac{2^2}{4}\right)=0.529\left(\dfrac{4}{4}\right)=0.529\)
        • This matches hydrogen exactly
      • Option (d): \((2),\ He^{+}\)
        • Here \(Z=2\)
        • \(r = 0.529\left(\dfrac{2^2}{2}\right)=0.529(2)\)
        • Not equal to \(0.529\)
    • Concept used / theory behind it
      • Hydrogen-like species are one-electron species such as \(He^+\), \(Li^{2+}\), \(Be^{3+}\), etc.
      • In Bohr’s model, orbit radius depends on both:
        • principal quantum number \(n\)
        • nuclear charge \(Z\)
      • The radius is directly proportional to \(n^2\) and inversely proportional to \(Z\).
      • So for equal radii, the quantity \(\dfrac{n^2}{Z}\) must be the same.
    • Why the correct option is right
      • For first orbit of hydrogen, \(\dfrac{n^2}{Z}=\dfrac{1^2}{1}=1\).
      • For \(Be^{3+}\) in the second orbit, \(\dfrac{n^2}{Z}=\dfrac{2^2}{4}=1\).
      • Both are equal, so both radii are equal.
    • Why other options are wrong
      • \((2),\ Li^{2+}\): gives \(\dfrac{4}{3}\), not \(1\).
      • \((3),\ Li^{2+}\): gives \(\dfrac{9}{3}=3\), not \(1\).
      • \((2),\ He^{+}\): gives \(\dfrac{4}{2}=2\), not \(1\).
    • Exam tip / shortcut / elimination clue
      • For equal Bohr radii, directly compare \(\dfrac{n^2}{Z}\).
      • For the first orbit of hydrogen, the target value is always \(1\).
      • So just look for the option where \(n^2 = Z\).
      • Here \(2^2=4\), so \(Be^{3+}\) fits immediately.
    • Final result
      • The required species is \(Be^{3+}\) and the orbit number is \(n=2\).
      • Hence, option (c) is correct.

    • Correct answer: (c) \(n=3,\ l=3,\ m=1,\ s=\dfrac{1}{2}\).
    • Explanation: For any principal quantum number \(n\), the azimuthal quantum number \(l\) can have values only from \(0\) to \(n-1\). When \(n=3\), possible \(l\) values are only \(0,\ 1,\ 2\). So \(l=3\) is not allowed, which makes this set impossible.
    • Step-by-step explanation
      • To test whether a quantum number set is possible, use the standard rules:
        • \(n = 1,2,3,\dots\)
        • \(l = 0\) to \(n-1\)
        • \(m = -l\) to \(+l\)
        • \(s = +\dfrac{1}{2}\) or \(-\dfrac{1}{2}\)
      • Now check each option.
      • Option (a): \(n=2,\ l=0,\ m=0,\ s=-\dfrac{1}{2}\)
        • For \(n=2\), allowed \(l\) values are \(0,1\)
        • If \(l=0\), then \(m=0\) is correct
        • Spin \(-\dfrac{1}{2}\) is also allowed
        • So this is possible
      • Option (b): \(n=2,\ l=1,\ m=0,\ s=\dfrac{1}{2}\)
        • For \(l=1\), allowed \(m\) values are \(-1,0,+1\)
        • So \(m=0\) is valid
        • Spin \(+\dfrac{1}{2}\) is valid
        • This set is possible
      • Option (c): \(n=3,\ l=3,\ m=1,\ s=\dfrac{1}{2}\)
        • For \(n=3\), allowed \(l\) values are only \(0,1,2\)
        • \(l=3\) is not possible at all
        • So the whole set becomes impossible immediately
      • Option (d): \(n=4,\ l=2,\ m=1,\ s=\dfrac{1}{2}\)
        • For \(n=4\), allowed \(l\) values are \(0,1,2,3\)
        • For \(l=2\), allowed \(m\) values are \(-2,-1,0,+1,+2\)
        • So \(m=1\) is valid
        • Spin \(+\dfrac{1}{2}\) is valid
        • This set is possible
    • Concept used / theory behind it
      • Each electron in an atom is described by four quantum numbers.
      • The most important restriction here is on \(l\).
      • If principal quantum number is \(n\), then:
        • \(l\) cannot be equal to \(n\)
        • \(l\) cannot be greater than \(n-1\)
      • For example:
        • if \(n=1\), \(l=0\) only
        • if \(n=2\), \(l=0,1\)
        • if \(n=3\), \(l=0,1,2\)
      • So \(n=3,\ l=3\) violates the basic rule.
    • Why the correct option is right
      • The pair \(n=3,\ l=3\) is invalid.
      • Since one quantum number is already impossible, the full set is impossible.
    • Why other options are wrong or less suitable
      • Options (a), (b), and (d) all satisfy the allowed ranges of \(n,\ l,\ m,\) and \(s\).
      • So they are possible quantum number sets.
    • Exam tip / shortcut / elimination clue
      • In questions like this, first check \(l\).
      • That eliminates wrong options fastest.
      • Remember:
        • \(l=0\) to \(n-1\)
      • Here \(n=3\) means \(l\) can only be \(0,1,2\), so option (c) is instantly rejected.
    • Final result
      • The impossible quantum number set is \(n=3,\ l=3,\ m=1,\ s=\dfrac{1}{2}\).
      • Hence, option (c) is correct.

    • Correct answer: (d) oxygen \(>\) fluorine \(>\) nitrogen \(>\) carbon.
    • Explanation: Second ionisation enthalpy means removing an electron from the singly charged cation. So we must compare \(C^+\), \(N^+\), \(O^+\), and \(F^+\). Among these, \(O^+\) has the configuration \(1s^2 2s^2 2p^3\), which is a half-filled stable configuration. Removing one more electron from this especially stable state requires maximum energy. On the other hand, \(C^+\) has configuration \(1s^2 2s^2 2p^1\), and losing one electron gives the stable filled \(2s^2\) configuration, so it requires the least energy.
    • Step-by-step explanation
      • Second ionisation enthalpy is the energy needed for:
        • \(X^+ \rightarrow X^{2+} + e^-\)
      • So do not look at the neutral atoms directly. First remove one electron from each neutral atom.
      • Electronic configurations after removal of first electron are:
        • \(C^+ = 1s^2\,2s^2\,2p^1\)
        • \(N^+ = 1s^2\,2s^2\,2p^2\)
        • \(O^+ = 1s^2\,2s^2\,2p^3\)
        • \(F^+ = 1s^2\,2s^2\,2p^4\)
      • Now compare how easy or difficult it is to remove one more electron from each of these cations.
      • \(O^+\) has \(2p^3\), which is a half-filled stable configuration. Breaking this stability needs very high energy.
      • \(F^+\) has \(2p^4\). Removing one electron from \(2p^4\) gives \(2p^3\), which is half-filled and more stable. So it is easier than for oxygen, but still not very easy because fluorine has higher nuclear charge.
      • \(N^+\) has \(2p^2\). Removing one electron gives \(2p^1\), which does not create any special stability. So its second ionisation enthalpy is lower than fluorine and oxygen.
      • \(C^+\) has \(2p^1\). Removing that electron gives \(1s^2 2s^2\), which is a completely filled subshell arrangement and hence relatively stable. So this removal is easiest among the four.
    • Concept used / theory behind it
      • Ionisation enthalpy generally increases across a period because effective nuclear charge increases.
      • But there are important exceptions due to:
        • half-filled stability
        • fully filled stability
        • electronic repulsion in paired orbitals
      • For higher ionisation enthalpies, always check the configuration of the ion from which the electron is being removed, not just the neutral atom.
      • That is the key trap in this question.
    • Why the correct option is right
      • \(O^+\) is highest because removing an electron breaks the stable half-filled \(2p^3\) configuration.
      • \(F^+\) is next because of high nuclear charge, though removal leads to a stable \(2p^3\) state.
      • \(N^+\) comes next.
      • \(C^+\) is lowest because electron removal leads to a more stable filled \(2s^2\) arrangement.
      • So the order is:
        • \(oxygen > fluorine > nitrogen > carbon\)
    • Why other options are wrong
      • Options placing carbon above nitrogen, oxygen, or fluorine ignore the special stability gained after \(C^+\) loses one electron.
      • Any option not keeping oxygen highest misses the fact that \(O^+\) is half-filled \((2p^3)\).
      • Any option placing fluorine below nitrogen also fails to account for its greater effective nuclear charge.
    • Exam tip / shortcut / elimination clue
      • For second ionisation enthalpy, always write the configuration of \(X^+\).
      • Then ask:
        • Is the electron being removed from a specially stable configuration?
        • Or does removal create a specially stable configuration?
      • Here:
        • \(O^+\) is hardest to ionise
        • \(C^+\) is easiest to ionise
      • That immediately narrows the answer to option (d).
    • Final result
      • The correct order of second ionisation enthalpies is:
        • oxygen \(>\) fluorine \(>\) nitrogen \(>\) carbon
      • Hence, option (d) is correct.

    • Correct answer: (c) I and IV only.
    • Explanation: \(CN^-\) and \(NO^+\) are isoelectronic species, each having 14 total electrons. Isoelectronic diatomic species generally have the same molecular orbital filling pattern and hence the same bond order. Therefore \(CN^-\) and \(NO^+\) have the same bond order.
    • Step-by-step explanation
      • Let us first count total electrons in each species.
      • I. \(CN^-\)
        • Carbon contributes \(6\) electrons
        • Nitrogen contributes \(7\) electrons
        • Negative charge adds \(1\) electron
        • Total \(= 6+7+1 = 14\)
      • II. \(N_2^+\)
        • Each nitrogen contributes \(7\) electrons
        • Total for \(N_2\) is \(14\)
        • Positive charge removes \(1\) electron
        • Total \(= 13\)
      • III. \(O_2^{2-}\)
        • Each oxygen contributes \(8\) electrons
        • Total for \(O_2\) is \(16\)
        • \(2-\) charge adds \(2\) electrons
        • Total \(= 18\)
      • IV. \(NO^+\)
        • Nitrogen contributes \(7\) electrons
        • Oxygen contributes \(8\) electrons
        • Positive charge removes \(1\) electron
        • Total \(= 7+8-1 = 14\)
      • So \(CN^-\) and \(NO^+\) are isoelectronic with \(14\) electrons each.
      • These species have the same molecular orbital electron arrangement and hence the same bond order.
      • In fact, both have bond order \(3\).
    • Concept used / theory behind it
      • Bond order from molecular orbital theory is:
        • \(\text{Bond order}=\dfrac{N_b-N_a}{2}\)
      • For isoelectronic diatomic species, the number of electrons is the same, so the MO filling pattern is usually the same or very closely comparable for standard exam problems.
      • That means their bond order also becomes the same.
      • This is a very common shortcut used in entrance examinations.
    • Why the correct option is right
      • \(CN^-\) has \(14\) electrons.
      • \(NO^+\) also has \(14\) electrons.
      • Therefore they have the same bond order.
      • So I and IV only is the correct pair.
    • Why other options are wrong
      • \(N_2^+\) has \(13\) electrons, so it is not isoelectronic with \(CN^-\) or \(NO^+\).
      • \(O_2^{2-}\) has \(18\) electrons, so it clearly does not match \(CN^-\) or \(NO^+\).
      • Hence the pairs involving II or III do not have the same bond order as the correct pair listed in option (c).
    • Exam tip / shortcut / elimination clue
      • For small diatomic ions, first count total electrons.
      • If two species are isoelectronic, they are often the correct pair for same bond order questions.
      • That saves time and avoids drawing full MO diagrams for every option.
    • Final result
      • The ions with the same bond order are \(CN^-\) and \(NO^+\).
      • Hence, option (c) I and IV only is correct.

    • Correct answer: (a) \(sp^2,\ sp,\ sp^3d\).
    • Explanation: \(BF_3\) has 3 electron domains around boron, so it is \(sp^2\) hybridised. \(BeF_2\) has 2 electron domains around beryllium, so it is \(sp\) hybridised. \(BrF_3\) has 5 electron domains around bromine \((3\ bond\ pairs + 2\ lone\ pairs)\), so it is \(sp^3d\) hybridised.
    • Step-by-step explanation
      • The simplest way is to count the number of electron pairs or electron domains around the central atom.
      • That number tells us the steric number, and from that we get the hybridisation.
      • For \(BF_3\):
        • Central atom is boron.
        • Boron forms 3 sigma bonds with 3 fluorine atoms.
        • There is no lone pair on boron.
        • So steric number \(= 3\).
        • Hybridisation \(= sp^2\).
      • For \(BeF_2\):
        • Central atom is beryllium.
        • Beryllium forms 2 sigma bonds with 2 fluorine atoms.
        • There is no lone pair on beryllium.
        • So steric number \(= 2\).
        • Hybridisation \(= sp\).
      • For \(BrF_3\):
        • Central atom is bromine.
        • Bromine has 7 valence electrons.
        • It forms 3 sigma bonds with 3 fluorine atoms.
        • That leaves 2 lone pairs on bromine.
        • Total electron domains \(= 3 + 2 = 5\).
        • So hybridisation \(= sp^3d\).
      • Hence the sequence is \(sp^2,\ sp,\ sp^3d\).
    • Concept used / theory behind it
      • Hybridisation depends on the steric number of the central atom.
      • General relation:
        • Steric number \(2 \rightarrow sp\)
        • Steric number \(3 \rightarrow sp^2\)
        • Steric number \(4 \rightarrow sp^3\)
        • Steric number \(5 \rightarrow sp^3d\)
      • In VSEPR language, we count both bond pairs and lone pairs because both occupy space around the central atom.
      • That is why \(BrF_3\), although it has only 3 bonded atoms, still has steric number 5 due to 2 lone pairs.
    • Why the correct option is right
      • \(BF_3 \rightarrow sp^2\)
      • \(BeF_2 \rightarrow sp\)
      • \(BrF_3 \rightarrow sp^3d\)
      • This matches option (a) exactly.
    • Why other options are wrong or less suitable
      • Option (b) wrongly gives \(BF_3\) as \(sp\), but boron has 3 electron domains, not 2.
      • Option (c) wrongly gives \(BF_3\) as \(sp^3\), but tetrahedral arrangement is not possible here.
      • Option (d) wrongly gives \(BeF_2\) as \(sp^3\), but beryllium in \(BeF_2\) has only 2 sigma bonds and no lone pairs.
      • \(BrF_3\) is not \(dsp^2\); its steric number is 5, so \(sp^3d\) is the standard description.
    • Exam tip / shortcut / elimination clue
      • Do not count only bonded atoms. Count total electron domains.
      • For quick recall:
        • \(BF_3\) is trigonal planar \(\rightarrow sp^2\)
        • \(BeF_2\) is linear \(\rightarrow sp\)
        • \(BrF_3\) is T-shaped with trigonal bipyramidal electron geometry \(\rightarrow sp^3d\)
    • Final result
      • The hybridisations are \(sp^2,\ sp,\ sp^3d\).
      • Hence, option (a) is correct.

    • Correct answer: (c) \(4 : 3\).
    • Explanation: By Graham’s law of diffusion, rate of diffusion is inversely proportional to the square root of molecular mass. So \(r \propto \dfrac{1}{\sqrt{M}}\). For gases of molecular masses 36 and 64, the ratio becomes \(r_X : r_Y = \dfrac{1}{\sqrt{36}} : \dfrac{1}{\sqrt{64}} = \dfrac{1}{6} : \dfrac{1}{8} = 8 : 6 = 4 : 3\).
    • Step-by-step explanation
      • Use Graham’s law:
        • \(r \propto \dfrac{1}{\sqrt{M}}\)
      • Here gas \(X\) has molecular mass \(36\), and gas \(Y\) has molecular mass \(64\).
      • So,
        • \(r_X \propto \dfrac{1}{\sqrt{36}} = \dfrac{1}{6}\)
        • \(r_Y \propto \dfrac{1}{\sqrt{64}} = \dfrac{1}{8}\)
      • Therefore,
        • \(\dfrac{r_X}{r_Y} = \dfrac{1/6}{1/8} = \dfrac{8}{6} = \dfrac{4}{3}\)
      • Hence the ratio \(r_X : r_Y = 4 : 3\).
    • Concept used / theory behind it
      • Graham’s law states that lighter gases diffuse faster than heavier gases.
      • This is because diffusion rate varies inversely with the square root of molar mass.
      • So if molecular mass increases, rate decreases.
      • That is why gas \(X\) with molecular mass 36 diffuses faster than gas \(Y\) with molecular mass 64.
    • Why the correct option is right
      • Direct substitution into Graham’s law gives \(4 : 3\).
      • This matches option (c).
    • Why other options are wrong or less suitable
      • Options using direct mass ratio are wrong because diffusion depends on square root, not the full molecular mass.
      • \(9:16\) and \(16:9\) come from incorrect handling of the numbers.
      • \(3:4\) is the reverse order and would incorrectly make the heavier gas diffuse faster.
    • Exam tip / shortcut / elimination clue
      • Always remember: lighter gas diffuses faster.
      • So the gas with mass 36 must have a higher rate than the gas with mass 64.
      • That immediately rules out any option where \(r_X < r_Y\).
      • Then take square roots:
        • \(\sqrt{36}=6,\ \sqrt{64}=8\)
      • Reverse them for rates: \(8:6 = 4:3\).
    • Final result
      • The ratio of rates of diffusion is \(4 : 3\).
      • Hence, option (c) is correct.

    • Explanation: Complete combustion of glucose is given by \(C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 + 6H_2O\). \(180\ g\) of glucose gives \(6\) moles of \(CO_2\) and \(6\) moles of \(H_2O\).     • Step-by-step explanation
      • First write the balanced combustion equation:
        • \(C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 + 6H_2O\)
      • Molar mass of glucose:
        • \((6 \times 12) + (12 \times 1) + (6 \times 16) = 72 + 12 + 96 = 180\ g\ mol^{-1}\)
      • Moles of \(10\ g\) glucose:
        • \(\dfrac{10}{180} = \dfrac{1}{18}\ mol\)
      • From the balanced equation:
        • \(1\ mol\) glucose gives \(6\ mol\ CO_2\)
        • \(1\ mol\) glucose gives \(6\ mol\ H_2O\)
      • So \(\dfrac{1}{18}\ mol\) glucose gives:
        • \(\dfrac{1}{18}\times 6 = \dfrac{1}{3}\ mol\ CO_2\)
        • \(\dfrac{1}{18}\times 6 = \dfrac{1}{3}\ mol\ H_2O\)
      • Now convert moles into mass.
      • Mass of \(CO_2\):
        • \(\dfrac{1}{3}\times 44 = 14.66\ g\)
      • Mass of \(H_2O\):
        • \(\dfrac{1}{3}\times 18 = 6.0\ g\)
    • Concept used / theory behind it
      • This is a standard stoichiometry problem.
      • In combustion reactions, always start with the balanced chemical equation.
      • Then convert:
        • given mass \(\rightarrow\) moles
        • moles of reactant \(\rightarrow\) moles of product
        • moles of product \(\rightarrow\) mass of product
      • The mention of STP does not affect the answer here because the question asks for masses, not gas volume.
    • Why the correct option is right
      • Balanced equation gives equal mole count of \(CO_2\) and \(H_2O\) as \(6\) each per mole of glucose.
      • For \(10\ g\) glucose, both products come out to \(\dfrac{1}{3}\ mol\).
      • That gives \(14.66\ g\) of \(CO_2\) and \(6.0\ g\) of \(H_2O\), matching option (b).
    • Why other options are wrong or less suitable
      • Option (a) gives correct \(CO_2\) mass but wrong \(H_2O\) mass.
      • Option (c) and option (d) do not follow the mole ratio from the balanced combustion equation.
      • A common mistake is using wrong molar mass or missing the factor of 6 for products.
    • Exam tip / shortcut / elimination clue

        • \(1\ mol\ C_6H_{12}O_6 \rightarrow 6\ mol\ CO_2 + 6\ mol\ H_2O\)
      • Once you find glucose moles as \(\dfrac{1}{18}\), multiplying by 6 quickly gives \(\dfrac{1}{3}\) mole of each product.

      • The masses formed are \(14.66\ g\) of \(CO_2\) and \(6.0\ g\) of \(H_2O\).

    • Correct answer: (a) \(-283.0\ kJ\ mol^{-1}\).
    • Explanation: Combustion of carbon monoxide is \(CO + \dfrac{1}{2}O_2 \rightarrow CO_2\). Using enthalpies of formation, \(\Delta H = \Sigma \Delta H_f(\text{products}) - \Sigma \Delta H_f(\text{reactants})\). So \(\Delta H = (-393) - (-110) = -283\ kJ\ mol^{-1}\).
    • Step-by-step explanation
      • First write the combustion reaction of carbon monoxide:
        • \(CO(g) + \dfrac{1}{2}O_2(g) \rightarrow CO_2(g)\)
      • Now apply the formula:
        • \(\Delta H_{reaction} = \Sigma \Delta H_f^\circ(\text{products}) - \Sigma \Delta H_f^\circ(\text{reactants})\)
      • Given:
        • \(\Delta H_f^\circ(CO) = -110\ kJ\ mol^{-1}\)
        • \(\Delta H_f^\circ(CO_2) = -393\ kJ\ mol^{-1}\)
        • \(\Delta H_f^\circ(O_2) = 0\), because \(O_2\) is in its standard elemental state
      • Substitute the values:
        • \(\Delta H = (-393) - \left( -110 + 0 \right)\)
        • \(\Delta H = -393 + 110\)
        • \(\Delta H = -283\ kJ\ mol^{-1}\)
      • So the enthalpy of combustion of \(CO\) is \(-283.0\ kJ\ mol^{-1}\).
    • Concept used / theory behind it
      • Standard enthalpy of formation is the enthalpy change when one mole of a compound is formed from its constituent elements in their standard states.
      • Standard enthalpy of combustion is the enthalpy change when one mole of a substance burns completely in oxygen.
      • These values are connected through Hess’s law:
        • \(\Delta H_{reaction} = \Sigma \Delta H_f^\circ(\text{products}) - \Sigma \Delta H_f^\circ(\text{reactants})\)
      • This is one of the most common thermochemistry formulas used in entrance exams.
    • Why the correct option is right
      • The product \(CO_2\) has formation enthalpy \(-393\).
      • The reactant \(CO\) has formation enthalpy \(-110\), while \(O_2\) contributes \(0\).
      • Therefore the combustion enthalpy becomes \(-393 - (-110) = -283\).
      • This matches option (a).
    • Why other options are wrong or less suitable
      • \(-110.5\) is not obtained from the combustion equation given.
      • \(504\) is not only numerically wrong, it also has the wrong sign since combustion is exothermic.
      • \(-221.2\) comes from incorrect subtraction.
    • Exam tip / shortcut / elimination clue
      • Combustion of \(CO\) to \(CO_2\) must be exothermic, so the answer must be negative.
      • Also, since \(CO_2\) is much more stable than \(CO\), the magnitude should be fairly large.
      • That quickly removes the positive option and helps you check the arithmetic.
    • Final result
      • The enthalpy of combustion of \(CO\) is \(-283.0\ kJ\ mol^{-1}\).
      • Hence, option (a) is correct.

        • \(X(g) + Y(g) \rightleftharpoons Z(g) + A(g)\)
      • Initial moles in a \(1\ L\) flask:
        • \(X = 1\)
        • \(Y = 1\)
        • \(Z = 0\)
        • \(A = 0\)
      • At equilibrium, \(0.5\ mol\) of \(Z\) is formed.
      • From the stoichiometry of the reaction:
        • 1 mol of \(Z\) formed means 1 mol of \(A\) formed
        • and 1 mol each of \(X\) and \(Y\) consumed
      • So if \(0.5\ mol\) of \(Z\) is formed:
        • \(A\) formed \(=0.5\ mol\)
        • \(X\) consumed \(=0.5\ mol\)
        • \(Y\) consumed \(=0.5\ mol\)
      • Equilibrium moles become:
        • \(X = 1 - 0.5 = 0.5\)
        • \(Y = 1 - 0.5 = 0.5\)
        • \(Z = 0 + 0.5 = 0.5\)
        • \(A = 0 + 0.5 = 0.5\)
      • Since volume \(=1\ L\), equilibrium concentrations are numerically the same:
        • \((X)=0.5,\ (Y)=0.5,\ (Z)=0.5,\ (A)=0.5\)
      • Now write the equilibrium constant expression:
        • \(K_C = \dfrac{(Z)(A)}{(X)(Y)}\)
      • Substitute values:
        • \(K_C = \dfrac{(0.5)(0.5)}{(0.5)(0.5)} = 1\)

      • This is a standard equilibrium mole-table question.
      • The safest method is:
        • write initial moles
        • write change in moles using stoichiometry
        • write equilibrium moles
        • substitute into \(K_C\)
      • Because the flask volume is \(1\ L\), there is no extra conversion needed from moles to molarity.
      • That makes the arithmetic very simple here.

        • \(1:1 \rightleftharpoons 1:1\)

      • A common mistake is to forget that \(A\) is also formed in the same amount as \(Z\).

      • \(K_C = 1.0\)
      • Hence, option (b) is correct.

    • Step-by-step explanation
      • This question is based on the stability of oxidation states in Group 14.
      • Group 14 elements have valence shell configuration:
        • \(ns^2 np^2\)
      • As we move down the group, the \(ns^2\) electrons become less willing to participate in bonding. This is called the inert pair effect.
      • In tin \((Sn)\), inert pair effect is present but not very strong.
      • So the \(+4\) oxidation state is still quite stable for tin.
      • That means \(Sn^{2+}\) can lose two more electrons to become \(Sn^{4+}\):
        • \(Sn^{2+} \rightarrow Sn^{4+} + 2e^-\)
      • Since it loses electrons, \(Sn^{2+}\) behaves as a reducing agent.
      • In lead \((Pb)\), inert pair effect is much stronger.
      • So the \(+2\) oxidation state is more stable than the \(+4\) state.
      • Therefore \(Pb^{4+}\) tends to gain two electrons and get reduced to \(Pb^{2+}\):
        • \(Pb^{4+} + 2e^- \rightarrow Pb^{2+}\)
      • Since it gains electrons, \(Pb^{4+}\) acts as an oxidising agent.

      • Inert pair effect is the reluctance of the outermost \(ns^2\) electron pair to participate in bonding.
      • This effect increases down the group because of poor shielding by inner \(d\) and \(f\) electrons and stronger effective nuclear attraction on the \(ns\) electrons.
      • As a result:
        • higher oxidation state becomes less stable down the group
        • lower oxidation state becomes more stable down the group
      • For Group 14:
        • \(C\) and \(Si\) prefer \(+4\)
        • \(Pb\) strongly prefers \(+2\)
      • This is why \(Pb^{4+}\) is a strong oxidising agent.
    • Why the correct option is right
      • \(Sn^{2+}\) tends to change into the more stable \(Sn^{4+}\), so it is a reducing agent.
      • \(Pb^{4+}\) tends to change into the more stable \(Pb^{2+}\), so it is an oxidising agent.
