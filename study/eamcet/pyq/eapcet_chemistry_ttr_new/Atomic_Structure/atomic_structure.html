Atomic_Structure

    Exam map from the PYQs
  • Most repeated cluster: hydrogen / hydrogen-like species using \(r_n \propto \dfrac{n^2}{Z}\) and \(E_n \propto -\dfrac{Z^2}{n^2}\).
  • Second major cluster: spectral-series identification, wavelength / wavenumber comparison, line counting, and energy-gap ratios.
  • Third major cluster: photoelectric effect — threshold condition, work function, \(K_{\max}\), speed, and “which metals emit / do not emit”.
  • Fourth cluster: quantum numbers, node formula, orbital count, \(m_l\), allowed / impossible sets, degeneracy ideas.
  • Fifth cluster: electronic configuration counting — \(s,p,d\) electron counts, \(n+l\) grouping, Cr / Cu exceptions, half-filled stability.
  • Smaller but recurring: de Broglie relation, uncertainty principle, Bohr angular momentum, orbital angular momentum.

    Hydrogen-like quick rules
  • Radius: \(r_n = \dfrac{n^2 a_0}{Z}\). Same radius \(\Rightarrow \dfrac{n^2}{Z}\) same.
  • Energy: \(E_n = -13.6\dfrac{Z^2}{n^2}\ \mathrm{eV}\). Same energy \(\Rightarrow \dfrac{Z^2}{n^2}\) same.
  • If radius is given, first identify \(n\) using \(r_n = a_0 n^2\) for H.
  • If species is \(\mathrm{He^+},\ \mathrm{Li^{2+}},\ \mathrm{Be^{3+}}\), immediately treat as one-electron species.
  • Same H \(1^\text{st}\) orbit radius type questions reduce to \( \dfrac{n^2}{Z} = 1\).
  • Same H \(2^\text{nd}\) orbit radius type questions reduce to \( \dfrac{n^2}{Z} = 4\).
  • Energy-gap questions: use \( \Delta E = 13.6Z^2\left(\dfrac{1}{n_1^2}-\dfrac{1}{n_2^2}\right)\) in magnitude.
  • Common trap: divide by \(Z\) for radius, but multiply by \(Z^2\) for energy magnitude.

    Spectral lines, series, wavelength, wavenumber
  • Rydberg form: \(\bar{\nu} = \dfrac{1}{\lambda} = RZ^2\left(\dfrac{1}{n_f^2}-\dfrac{1}{n_i^2}\right)\).
  • Lyman \(\Rightarrow n_f = 1\), Balmer \(\Rightarrow n_f = 2\), Paschen \(\Rightarrow n_f = 3\), Pfund \(\Rightarrow n_f = 5\).
  • If final orbit radius gives \(n=3\), it is Paschen immediately.
  • “First line” means \(n_i = n_f + 1\); “second line” means \(n_i = n_f + 2\); “third line” means \(n_i = n_f + 3\).
  • Total lines between two shells: if all possible downward transitions are allowed among \(n_2\) to \(n_1\), use \( \dfrac{m(m-1)}{2}\) with \(m = n_2 - n_1 + 1\).
  • For hydrogen, energy-gap ratios are faster from \( \dfrac{1}{n^2}\) values than from decimal energies.
  • As \(n\) increases, levels get closer; upper-level gaps are smaller.
  • Common trap: wavelength and wavenumber are inverse; photon energy is inversely proportional to wavelength, directly proportional to wavenumber.

    Photoelectric effect solve-fast sheet
  • Core equation: \(K_{\max} = h\nu - \phi = \dfrac{hc}{\lambda} - \phi\).
  • Threshold frequency: \(\nu_0 = \dfrac{\phi}{h}\). Threshold wavelength: \(\lambda_0 = \dfrac{hc}{\phi}\).
  • Emission occurs only if \(h\nu \ge \phi\) or \( \dfrac{1240}{\lambda(\mathrm{nm})} \ge \phi(\mathrm{eV})\).
  • Fast shortcut: \(E(\mathrm{eV}) = \dfrac{1240}{\lambda(\mathrm{nm})}\).
  • Given \(K_{\max}\), electron speed: \(v = \sqrt{\dfrac{2K_{\max}}{m_e}}\).
  • Intensity controls number of electrons, not \(K_{\max}\), if frequency is above threshold.
  • No appreciable time lag is a true photoelectric-effect hallmark.
  • When options give several metals, compute photon energy once, then compare with all work functions.
  • Common trap: students compare wavelength directly instead of converting to photon energy.

    Quantum numbers, orbitals, nodes
  • Allowed values: \(n=1,2,3,\dots\); \(l = 0 \text{ to } n-1\); \(m_l = -l \text{ to } +l\); \(m_s = \pm \dfrac12\).
  • Impossible set check: first test whether \(l \le n-1\).
  • Number of orbitals in a subshell \(= 2l+1\).
  • Total orbitals in shell \(n = n^2\); maximum electrons in shell \(= 2n^2\).
  • Maximum number of electrons with same spin in shell \(n\) equals number of orbitals \(= n^2\).
  • Radial nodes \(= n-l-1\); angular nodes \(= l\); total nodes \(= n-1\).
  • In H-atom, same \(n\) orbitals like \(2s\) and \(2p\) are degenerate; in multi-electron atoms they are not.
  • \(m_l\) gives orbital orientation; \(\psi\) stores full wave information; probability density is \(|\psi|^2\), not \(|\psi|^3\).

    Electronic configuration and counting shortcuts
  • Use Aufbau order, but memorize exceptions: \(\mathrm{Cr} = 3d^5 4s^1\), \(\mathrm{Cu} = 3d^{10} 4s^1\).
  • Half-filled and completely filled subshells are extra stable.
  • \(\mathrm{IE_1(N) > IE_1(O)}\) is a must-memorize exception due to \(2p^3\) stability.
  • \(s\)-electron / \(p\)-electron ratio questions need total occupied \(s\) and \(p\) electrons, not valence only unless stated.
  • \((n+l)\) grouping questions: write configuration fully, then tag each occupied subshell by its \(n+l\) value.
  • Isoelectronic species: compare total electrons after charge adjustment, not atomic numbers alone.

    de Broglie, uncertainty, angular momentum
  • de Broglie: \(\lambda = \dfrac{h}{p} = \dfrac{h}{mv}\).
  • From wave relation \( \lambda = \dfrac{c}{\nu}\), particle speed form becomes \(v = \dfrac{h\nu}{mc}\).
  • Heisenberg: \(\Delta x\,\Delta p \ge \dfrac{h}{4\pi}\), with \(\Delta p = m\Delta v\).
  • Speed-%-uncertainty questions: first find \(\Delta v\), then \( \dfrac{\Delta v}{v}\times 100\).
  • Bohr angular momentum: \(mvr = \dfrac{nh}{2\pi}\); ground-state hydrogen gives \( \dfrac{h}{2\pi}\).
  • Orbital angular momentum magnitude: \( \sqrt{l(l+1)}\,\hbar\); for \(d\)-orbital, \(l=2 \Rightarrow \sqrt{6}\hbar\).

    Common traps from the PYQs
  • Do not apply hydrogen degeneracy blindly to He, Li, or other multi-electron atoms.
  • Do not forget that \(\mathrm{He^+}\), \(\mathrm{Li^{2+}}\), \(\mathrm{Be^{3+}}\) are hydrogen-like, but \(\mathrm{He}\) is not.
  • For spectral-series questions, identify the final shell first; that decides the series name.
  • For photoelectric questions, compare photon energy with work function before doing anything else.
  • For quantum-number validity, invalid \(l\) is the fastest elimination trigger.
  • For electron-count questions, use actual ground-state configuration, including Cr / Cu exceptions.

    Atomic models and why Bohr was needed
  • Thomson model: atom treated as uniformly positive sphere with embedded electrons; could not explain scattering results.
  • Rutherford model: atom has tiny dense positively charged nucleus; most space is empty; explains \(\alpha\)-particle scattering.
  • Main drawback of Rutherford model: revolving electron should continuously radiate energy and collapse into nucleus, so atomic stability is unexplained.
  • Bohr fixed this by postulating stationary orbits with fixed energies and no radiation during orbiting.
  • Radiation is emitted or absorbed only during transitions between allowed levels.

    Electromagnetic radiation and Planck quantum idea
  • Electromagnetic radiation is characterized by wavelength \(\lambda\), frequency \(\nu\), wavenumber \(\bar{\nu}\), and velocity \(c\).
  • Basic relations: \(c = \nu\lambda\), \(\bar{\nu} = \dfrac{1}{\lambda}\).
  • Photon energy: \(E = h\nu = \dfrac{hc}{\lambda} = hc\bar{\nu}\).
  • As wavelength decreases, frequency, wavenumber, and photon energy increase.
  • Planck quantum theory: energy is emitted or absorbed in discrete packets called quanta; for light, those packets are photons.
  • This is the base idea behind photoelectric-effect questions and wavelength-to-energy conversion.
  • Useful exam shortcut: \(E(\mathrm{eV}) = \dfrac{1240}{\lambda(\mathrm{nm})}\).

    Bohr model for hydrogen and hydrogen-like species
  • Applicable to one-electron species: \(\mathrm{H}, \mathrm{He^+}, \mathrm{Li^{2+}}, \mathrm{Be^{3+}}\), etc.
  • Allowed radius of \(n^\text{th}\) orbit: \(r_n = \dfrac{n^2 a_0}{Z}\), where \(a_0 = 0.529\ \text{\AA} = 52.9\ \mathrm{pm}\).
  • Energy of \(n^\text{th}\) orbit: \(E_n = -13.6\dfrac{Z^2}{n^2}\ \mathrm{eV}\).
  • Bohr angular momentum quantization: \(mvr = \dfrac{nh}{2\pi}\).
  • For hydrogen, \(r_n \propto n^2\), \(E_n \propto -\dfrac{1}{n^2}\).
  • With increasing \(n\), orbit radius increases, energy becomes less negative, and spacing between levels decreases.
  • If question says same radius, compare \( \dfrac{n^2}{Z}\). If same energy, compare \( \dfrac{Z^2}{n^2}\).
  • If orbit radius is given, recover \(n\) first; if energy is asked, identify \(Z\) or \(n\) first.

    Hydrogen spectrum, series, and transition logic
  • Rydberg equation: \( \dfrac{1}{\lambda} = RZ^2\left(\dfrac{1}{n_f^2} - \dfrac{1}{n_i^2}\right)\), where \(n_i > n_f\).
  • Lyman series: \(n_f = 1\), ultraviolet region.
  • Balmer series: \(n_f = 2\), visible region.
  • Paschen series: \(n_f = 3\), infrared region.
  • “First line” means \(n_i = n_f + 1\); “second line” means \(n_i = n_f + 2\); “third line” means \(n_i = n_f + 3\).
  • Number of possible spectral lines among \(m\) levels is \( \dfrac{m(m-1)}{2}\).
  • Energy of emitted photon in transition \(n_i \to n_f\): \( \Delta E = 13.6Z^2\left(\dfrac{1}{n_f^2} - \dfrac{1}{n_i^2}\right)\ \mathrm{eV}\).

    Photoelectric effect
  • Einstein equation: \(K_{\max} = h\nu - \phi\), where \(\phi\) is work function.
  • Threshold frequency: \(\nu_0 = \dfrac{\phi}{h}\); threshold wavelength: \(\lambda_0 = \dfrac{hc}{\phi}\).
  • No photoemission occurs if \(h\nu < \phi\).
  • Maximum kinetic energy depends on frequency, not intensity.
  • Number of emitted electrons depends mainly on intensity, provided frequency is above threshold.
  • There is essentially no time lag between incidence and emission in ordinary photoelectric effect statements.
  • Electron speed after emission: \(v = \sqrt{\dfrac{2K_{\max}}{m_e}}\).
  • For metal-comparison PYQs, calculate photon energy once, then compare with each work function.

    Dual nature of matter and uncertainty principle
  • de Broglie proposed that moving particles have wave nature.
  • de Broglie wavelength: \(\lambda = \dfrac{h}{p} = \dfrac{h}{mv}\).
  • Combining \( \lambda = \dfrac{c}{\nu}\) and \( \lambda = \dfrac{h}{mv}\) gives \(v = \dfrac{h\nu}{mc}\).
  • Heisenberg uncertainty principle: \( \Delta x\, \Delta p \ge \dfrac{h}{4\pi}\).
  • Since \( \Delta p = m\Delta v\), we also use \( \Delta x\, m\Delta v \ge \dfrac{h}{4\pi}\).
  • For microscopic particles, uncertainty is significant; for macroscopic bodies, it is negligible.
  • Percentage uncertainty in speed \(= \dfrac{\Delta v}{v}\times 100\).

    Quantum mechanical model and wave function
  • Bohr model works well for one-electron species but fails for many-electron atoms, fine details, and wave nature.
  • Quantum mechanics treats electron through wave function \(\psi\), not as a fixed classical path.
  • \(\psi\) itself has no direct physical meaning; \(|\psi|^2\) gives probability density.
  • An orbital is a region in space with high probability of finding an electron.
  • Quantum mechanics explains quantized energies, orbital shapes, and electron probability distribution more completely than Bohr model.

    Quantum numbers and what each one means
  • Principal quantum number \(n\): shell, size, and rough energy level.
  • Azimuthal quantum number \(l\): subshell and orbital shape; \(l=0,1,2,3\) correspond to \(s,p,d,f\).
  • Magnetic quantum number \(m_l\): orbital orientation; values from \(-l\) to \(+l\).
  • Spin quantum number \(m_s\): electron spin orientation; \(+\dfrac12\) or \(-\dfrac12\).
  • Allowed-value rule: for given \(n\), \(l = 0\) to \(n-1\).
  • Number of orbitals in subshell \(= 2l+1\); number of orbitals in shell \(= n^2\).
  • Maximum electrons in shell \(= 2n^2\).
  • Maximum same-spin electrons in a shell \(= n^2\), one per orbital.

    Shapes, nodes, and energies of orbitals
  • \(s\)-orbitals are spherical; \(p\)-orbitals are dumbbell-shaped; \(d\)-orbitals are clover-like except \(d_{z^2}\).
  • Total nodes \(= n-1\).
  • Angular nodes \(= l\).
  • Radial nodes \(= n-l-1\).
  • For H atom, all orbitals with same \(n\) have same energy.
  • For multi-electron atoms, energies depend on both \(n\) and \(l\); generally \(s < p < d < f\) within same shell.
  • In H atom, \(2s\) and \(2p\) are degenerate; in He or other many-electron atoms they are not.

    Filling of orbitals and electronic configuration
  • Aufbau principle: electrons fill lower-energy orbitals first.
  • \((n+l)\) rule: lower \((n+l)\) fills first; if equal, lower \(n\) fills first.
  • Pauli exclusion principle: no two electrons in an atom have all four quantum numbers identical.
  • Hund rule: in degenerate orbitals, electrons occupy singly with parallel spins before pairing.
  • Ground-state configurations often solve counting PYQs faster than formula-based methods.
  • Subshell capacities: \(s=2,\ p=6,\ d=10,\ f=14\).

    Special stability and common exceptions
  • Half-filled and completely filled subshells have extra stability due to symmetry and exchange energy considerations.
  • Chromium: expected \(3d^4 4s^2\), actual \(3d^5 4s^1\).
  • Copper: expected \(3d^9 4s^2\), actual \(3d^{10} 4s^1\).
  • Nitrogen \(2p^3\) is extra stable; oxygen \(2p^4\) has one paired \(p\)-electron and slightly lower first ionisation enthalpy.
  • These exceptions repeatedly appear in configuration, counting, and assertion-reason questions.

    High-yield question-to-formula mapping
  • If question asks same radius \(\Rightarrow\) use \( \dfrac{n^2}{Z}\).
  • If question asks same energy \(\Rightarrow\) use \( \dfrac{Z^2}{n^2}\).
  • If wavelength / wavenumber of spectral line is asked \(\Rightarrow\) use Rydberg formula.
  • If series name is asked \(\Rightarrow\) first identify final orbit \(n_f\).
  • If line count is asked \(\Rightarrow\) use \( \dfrac{m(m-1)}{2}\).
  • If threshold / work function / emitted speed is asked \(\Rightarrow\) use photoelectric equation first.
  • If quantum-number validity is asked \(\Rightarrow\) check \(l \le n-1\) before anything else.
  • If counting electrons by type is asked \(\Rightarrow\) write full ground-state configuration first.
