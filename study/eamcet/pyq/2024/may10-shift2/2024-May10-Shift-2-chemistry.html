    Answer: \((a)\ 12.5\%,\ 50\%\)

    Solution:

    Given crystal formula:

    \(AB_2O_4\)

    Oxygen atoms form a ccp lattice.

    1. First count the number of oxygen atoms in one ccp unit cell

      In a ccp arrangement, the effective number of atoms per unit cell is:

      \(4\)

      So, in one unit cell, the number of oxygen atoms is \(4\).

    2. Use the formula \(AB_2O_4\) to find how many \(A\) and \(B\) atoms are present

      The ratio is:

      \(A:B:O = 1:2:4\)

      Since there are \(4\) oxygen atoms in one unit cell, the corresponding numbers are:

      • \(A = 1\)
      • \(B = 2\)
      • \(O = 4\)
    3. Now count the voids in ccp

      If the number of close-packed particles is \(n\), then:

      • Number of tetrahedral voids \(= 2n\)
      • Number of octahedral voids \(= n\)

      Here \(n = 4\), so:

      • Number of tetrahedral voids \(= 8\)
      • Number of octahedral voids \(= 4\)
    4. Find \(x\%\): fraction of tetrahedral voids occupied by \(A\)

      Number of \(A\) atoms \(= 1\)

      Total tetrahedral voids \(= 8\)

      Therefore,

      \(x = \dfrac{1}{8}\times 100 = 12.5\%\)

    5. Find \(y\%\): fraction of octahedral voids occupied by \(B\)

      Number of \(B\) atoms \(= 2\)

      Total octahedral voids \(= 4\)

      Therefore,

      \(y = \dfrac{2}{4}\times 100 = 50\%\)

    Therefore:

    \(\boxed{x = 12.5\%,\ y = 50\%}\)

    \(\boxed{(a)\ 12.5\%,\ 50\%}\)

    Why this method works:

    • In close packing questions, the whole idea is to connect:

      \(\text{number of host atoms} \rightarrow \text{number of voids} \rightarrow \text{fraction occupied}\)

    • Since oxygen forms ccp, oxygen is the host lattice.
    • Once the number of oxygen atoms per unit cell is known, the formula \(AB_2O_4\) immediately tells us how many \(A\) and \(B\) atoms must be present.
    • Then we compare those numbers with the available tetrahedral and octahedral voids.

    JEE / NCERT takeaway:

    • In ccp/fcc arrangement, effective number of particles per unit cell is:

      \(4\)

    • If close-packed particles \(= n\), then:
      • Tetrahedral voids \(= 2n\)
      • Octahedral voids \(= n\)
    • Therefore, for ccp with \(4\) host atoms:
      • Tetrahedral voids \(= 8\)
      • Octahedral voids \(= 4\)
    • This is one of the most important solid state results for numericals based on void occupancy.
    • In such questions, the non-close-packed ions are usually placed in tetrahedral or octahedral voids according to the given formula.

    1-minute solving trick:

    • The moment you see oxygen forms ccp, write directly:

      \(O = 4,\quad \text{tetrahedral voids} = 8,\quad \text{octahedral voids} = 4\)

    • From \(AB_2O_4\), directly match with \(O=4\):

      \(A = 1,\quad B = 2\)

    • Then:

      \(x = \dfrac{1}{8}\times 100 = 12.5\%\)

      \(y = \dfrac{2}{4}\times 100 = 50\%\)

    • This is a very fast pattern-based question if the void formulas are remembered.

    More intuitive memory trick:

    • For close packing:

      \(\text{octahedral voids} = \text{host atoms}\)

    • \(\text{tetrahedral voids} = 2 \times \text{host atoms}\)

    • So if host atoms are \(4\), then just think:

      \(4 \rightarrow 4\ \text{octahedral},\ 8\ \text{tetrahedral}\)

    • That single memory line solves many EAPCET/JEE void problems.

    Common mistakes to avoid:

    1. Forgetting that ccp and fcc have \(4\) particles per unit cell.
    2. Using wrong void counts:

      \(\text{tetrahedral voids} \neq n,\quad \text{they are }2n\)

    3. Mixing up which atom occupies which void.
    4. Taking percentages using total atoms instead of total voids of that type.
    5. Forgetting to first match the formula with the \(4\) oxygen atoms in the unit cell.

    Final exam-style answer:

    Oxygen atoms form ccp arrangement, so the number of oxygen atoms per unit cell is \(4\).

    Hence in one unit cell, from \(AB_2O_4\):

    \(A = 1,\quad B = 2,\quad O = 4\)

    For ccp arrangement of \(4\) oxygen atoms:

    Number of tetrahedral voids \(= 8\)

    Number of octahedral voids \(= 4\)

    Since \(A\) occupies tetrahedral voids,

    \(x = \dfrac{1}{8}\times 100 = 12.5\%\)

    Since \(B\) occupies octahedral voids,

    \(y = \dfrac{2}{4}\times 100 = 50\%\)

    \(\boxed{(a)\ 12.5\%,\ 50\%}\)

    Quick cheatsheet:

    • ccp/fcc particles per unit cell:

      \(4\)

    • If host atoms \(= n\):

      \(\text{tetrahedral voids} = 2n\)

      \(\text{octahedral voids} = n\)

    • For ccp with \(4\) host atoms:

      \(\text{tetrahedral voids} = 8,\quad \text{octahedral voids} = 4\)

    • Formula matching trick:

      Match compound formula with the number of host lattice atoms first.

    • Percent occupancy:

      \(\dfrac{\text{number occupied}}{\text{total voids of that type}}\times 100\)
