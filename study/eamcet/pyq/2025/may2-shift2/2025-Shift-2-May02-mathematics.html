    • Answer: (a) 84
    1. Step-by-step intuition (clear understanding)
      We are working with a 4-digit number, so let its digits be \(a, b, c, d\).
    2. Maximum possible digit sum
      Since each digit can be at most \(9\), the maximum possible sum is
      \(9 + 9 + 9 + 9 = 36\)
    3. Given condition
      We are given that
      \(a + b + c + d = 30\)
    4. Key idea: think in reverse
      Instead of trying to build the sum \(30\) directly, start from the maximum possible sum \(36\) and see how much less is needed.
      So the total reduction required is
      \(36 - 30 = 6\)
    5. Meaning of this reduction
      Think of each digit as starting from \(9\), and then being reduced by some amount. Let
      • \(x_1 = 9 - a\)
      • \(x_2 = 9 - b\)
      • \(x_3 = 9 - c\)
      • \(x_4 = 9 - d\)
      Then:
      • if a digit is \(9\), its reduction is \(0\)
      • if a digit is \(8\), its reduction is \(1\)
      • if a digit is \(7\), its reduction is \(2\)
      • and so on
    6. New equation formed
      Since total reduction is \(6\), we get
      \(x_1 + x_2 + x_3 + x_4 = 6\)
      where each \(x_i \ge 0\)
    7. Interpretation of the new equation
      Now the question becomes:
      In how many ways can total reduction \(6\) be distributed among 4 digits?
      This is the standard problem of counting the number of non-negative integral solutions of
      \(x_1 + x_2 + x_3 + x_4 = 6\)
    8. Apply the standard result
      Number of non-negative integral solutions of
      \(x_1 + x_2 + x_3 + x_4 = 6\)
      is
      \(\binom{6+4-1}{4-1} = \binom{9}{3} = 84\)
    9. Why this automatically gives valid 4-digit numbers
      We should check whether the first digit can become \(0\).
      If \(a = 0\), then \(x_1 = 9\). But that is impossible because the total of all reductions is only \(6\).
      So \(a \ge 1\) automatically, which means every solution obtained is a valid 4-digit number.
    10. Final answer
      Therefore, the number of required integers is
      \(\boxed{84}\)
    1. Intuition summary
      Instead of asking:
      “How do I make digit sum \(30\)?”
      ask:
      “How far is \(30\) from the maximum possible sum \(36\)?”
      The gap is \(6\), so we only need to distribute this shortfall among the 4 digits.
    2. Trick for future questions
      Use this whenever:
      • the number of digits is fixed, and
      • the required digit sum is close to the maximum possible sum
    3. General trick
      For an \(n\)-digit number:
      • maximum digit sum \(= 9n\)
      • if required sum \(= S\), then deficit \(= 9n - S\)
      Then count the number of non-negative integral solutions of
      \(x_1 + x_2 + \dots + x_n = 9n - S\)
      using
      \(\binom{(9n-S)+n-1}{n-1}\)
    4. Apply that trick here
      For this question:
      • number of digits \(= 4\)
      • maximum sum \(= 36\)
      • required sum \(= 30\)
      • deficit \(= 36 - 30 = 6\)
      So answer \(= \binom{6+3}{3} = \binom{9}{3} = 84\)
    5. One-line exam method
      Deficit from maximum sum \(= 36 - 30 = 6\).
      Number of non-negative integral solutions \(= \binom{9}{3} = 84\).

      • Subtopic: Digit-sum problems using deficit method
