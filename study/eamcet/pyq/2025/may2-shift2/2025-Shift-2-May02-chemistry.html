    1. Step 1: Convert units
      \(476.1 \text{ pm} = 4.761 \text{ Å}\)
    2. Step 2: Use Bohr radius formula
      For hydrogen atom: \[ r_n = a_0 n^2 = 0.529\, n^2 \text{ Å} \]
    3. Step 3: Solve for \(n\)
      \[ 4.761 = 0.529\, n^2 \Rightarrow n^2 = \frac{4.761}{0.529} = 9 \Rightarrow n = 3 \]
    4. Step 4: Identify spectral series
      The given orbit is the final orbit (electron falls to this orbit).
      Series classification:
      • Lyman: \(n_f = 1\) (UV region)
      • Balmer: \(n_f = 2\) (visible region)
      • Paschen: \(n_f = 3\) (infrared region)
      • Brackett: \(n_f = 4\)
      • Pfund: \(n_f = 5\)
      Since \(n_f = 3\), the transition belongs to Paschen series.
    1. Core Theory: Bohr Model (NCERT Standard)
      • Electrons revolve in fixed circular orbits without radiating energy.
      • Allowed orbits are defined by quantized angular momentum: \[ mvr = \frac{nh}{2\pi} \]
      • Radius of \(n^{th}\) orbit: \[ r_n = a_0 n^2 \quad (\text{for H atom}) \]
      • Energy of orbit: \[ E_n = -\frac{13.6}{n^2} \text{ eV} \]
      • As \(n\) increases:
        • Radius increases (\(\propto n^2\))
        • Energy increases (less negative)
        • Electron moves farther from nucleus
    2. Electronic Transitions & Spectrum
      • When electron falls from higher orbit \(n_2\) to lower orbit \(n_1\), energy is released: \[ \Delta E = h\nu = \frac{hc}{\lambda} \]
      • Rydberg equation: \[ \frac{1}{\lambda} = R\left(\frac{1}{n_1^2} - \frac{1}{n_2^2}\right) \]
      • The series depends only on final orbit \(n_1\), not initial orbit.
    3. Hydrogen Spectral Series (Must Memorize)
      • Lyman (\(n=1\)) → Ultraviolet
      • Balmer (\(n=2\)) → Visible
      • Paschen (\(n=3\)) → Infrared
      • Brackett (\(n=4\)) → Infrared
      • Pfund (\(n=5\)) → Infrared
    4. Important Patterns (JEE/EAPCET)
      • Radius ∝ \(n^2\)
      • Energy ∝ \(-1/n^2\)
      • Gap between levels decreases as \(n\) increases
      • Longest wavelength → smallest energy transition in that series
    5. Fast Solving Tricks
      • Convert pm → Å quickly: divide by 100
      • Use: \[ r_n = 0.529 n^2 \]
      • Recognize common values:
        • \(n=1\): \(0.529\) Å
        • \(n=2\): \(2.116\) Å
        • \(n=3\): \(4.761\) Å
        • \(n=4\): \(8.464\) Å
      • If value matches → directly identify \(n\)
      • Then map:
        • 1 → Lyman
        • 2 → Balmer
        • 3 → Paschen
    6. Common Mistakes
      • Using initial orbit instead of final orbit
      • Forgetting unit conversion
      • Mixing up Balmer (visible) and Paschen (IR)

    • Answer: (b) is incorrect
    1. Question transcription
      Identify the incorrect statement from the following:
      • (a) \(m_l\) designates the orientation of the orbital.
      • (b) The probability density of electron is expressed by \(|\psi|^3\).
      • (c) The total information about electron in atom is stored in its \(\psi\).
      • (d) Total number of orbitals in a sub level is equal to \(2l + 1\).
    2. Why option (b) is incorrect
      Option (b) says that the probability density is expressed by \(|\psi|^3\), but this is wrong.
      The correct expression is
      \(|\psi|^2\)
      In quantum mechanics:
      • \(\psi\) is the wave function
      • \(|\psi|^2\) gives the probability density of finding the electron in space
      Therefore, option (b) is the incorrect statement.
    3. Checking option (a)
      \(m_l\) designates the orientation of the orbital. This statement is correct.
      The magnetic quantum number \(m_l\) tells us how an orbital is oriented in space.
      • For a given \(l\), \(m_l\) can have values from \(-l\) to \(+l\)
      • Different values of \(m_l\) correspond to different orbitals of the same subshell
      Example:
      • For \(l=1\) (p-subshell), \(m_l = -1, 0, +1\)
      • So there are three p orbitals with different orientations
    4. Checking option (b)
      The statement says probability density is \(|\psi|^3\). This is incorrect.
      The correct form is
      \(|\psi|^2\)
      This is one of the most important standard facts from the quantum mechanical model of atom.
    5. Checking option (c)
      “The total information about electron in atom is stored in its \(\psi\)” is correct.
      The wave function \(\psi\) is the mathematical description of the electron state. It contains the complete information about the state of the electron.
      However, an important NCERT point is:
      • \(\psi\) itself has no direct physical significance
      • \(|\psi|^2\) has direct physical meaning as probability density
      These two ideas do not contradict each other.
    6. Checking option (d)
      “Total number of orbitals in a sub level is equal to \(2l + 1\)” is correct.
      For a given value of \(l\), the magnetic quantum number \(m_l\) takes values:
      \(-l, -(l-1), \dots, 0, \dots, +(l-1), +l\)
      Hence total number of possible values is
      \(2l + 1\)
      Since each value of \(m_l\) corresponds to one orbital, total number of orbitals in a subshell is also
      \(2l + 1\)
    7. Examples of number of orbitals in subshells
      • s-subshell: \(l=0\), so number of orbitals \(= 2(0)+1 = 1\)
      • p-subshell: \(l=1\), so number of orbitals \(= 2(1)+1 = 3\)
      • d-subshell: \(l=2\), so number of orbitals \(= 2(2)+1 = 5\)
      • f-subshell: \(l=3\), so number of orbitals \(= 2(3)+1 = 7\)
    8. Final answer
      The incorrect statement is
      (b)
      because probability density is expressed by
      \(|\psi|^2\)
      and not by \(|\psi|^3\)
    1. Detailed theory: wave function \(\psi\)
      In the quantum mechanical model, the electron is not treated as moving in a fixed circular path. Instead, its state is described by a mathematical function called the wave function, written as \(\psi\).
      This wave function is obtained by solving the Schrödinger equation.
    2. Physical significance of \(\psi\) and \(|\psi|^2\)
      • \(\psi\) itself has no direct physical significance
      • \(|\psi|^2\) gives the probability density of finding the electron at a point in space
      • If \(|\psi|^2\) is large in a region, the chance of finding the electron there is high
      This is a standard NCERT and JEE point.
    3. Quantum numbers involved in atomic structure
      An electron in an atom is described by four quantum numbers.
      • Principal quantum number \(n\):
        tells the main shell or energy level
      • Azimuthal quantum number \(l\):
        tells the subshell and shape of orbital
        • \(l=0 \rightarrow s\)
        • \(l=1 \rightarrow p\)
        • \(l=2 \rightarrow d\)
        • \(l=3 \rightarrow f\)
      • Magnetic quantum number \(m_l\):
        tells the orientation of orbital in space
      • Spin quantum number \(m_s\):
        tells the spin of the electron, with values \(+\frac{1}{2}\) or \(-\frac{1}{2}\)
    4. Why \(m_l\) is connected with orientation
      Orbitals of the same subshell have the same shape and energy in many cases, but they differ in their orientation in space. This difference is represented by the magnetic quantum number \(m_l\).
      So whenever an option says \(m_l\) gives orientation, it is a standard correct statement.
    5. Why number of orbitals is \(2l + 1\)
      For a given \(l\), \(m_l\) can take all integer values from \(-l\) to \(+l\). Therefore, total number of possible values is
      \(2l + 1\)
      Since each value of \(m_l\) corresponds to one orbital, the number of orbitals in that subshell is also
      \(2l + 1\)
    6. Important NCERT/JEE distinction
      Students often confuse these two statements:
      • \(\psi\) contains the information about the electron state
      • \(\psi\) has no direct physical significance
      Both are true.
      The wave function is the mathematical description of the electron state, but the directly meaningful quantity is \(|\psi|^2\), not \(\psi\) itself.
    7. One-minute exam tip
      This question can be solved very quickly by spotting one standard fact:
      Probability density is always \(|\psi|^2\)
      So if you see \(|\psi|^3\), it is immediately wrong.
    8. Fast memory anchors
      • \(m_l\) → orientation
      • \(|\psi|^2\) → probability density
      • \(2l+1\) → number of orbitals in a subshell
      These three facts alone are enough to solve this question in under a minute.
    9. Common confusion to avoid
      • Do not confuse \(\psi\) with \(|\psi|^2\)
      • Do not think that “no direct physical significance” means \(\psi\) is useless; it still describes the state mathematically
      • Do not forget that \(2l+1\) gives the number of orbitals in a subshell, not the number of electrons
    10. Topic classification
      • Chapter: Structure of Atom
      • Topic: Quantum mechanical model of atom
      • Subtopic: Wave function, probability density, quantum numbers, number of orbitals in a subshell
    11. Super-short exam version
      • \(m_l\) gives orientation → true
      • Probability density is \(|\psi|^2\), not \(|\psi|^3\) → false statement is (b)
      • \(\psi\) stores information about electron state → true
      • Number of orbitals in a subshell \(= 2l+1\) → true
      Therefore, the incorrect statement is (b).
