    • Step-by-step explanation
      • Start with ideal gas equation:
        • \(pV = nRT\)
      • For an isobar, pressure remains constant.
      • So rearranging:
        • \(V = \dfrac{nRT}{p}\)
      • This can be written in the form:
        • \(V = \left(\dfrac{nR}{p}\right)T\)
      • Compare this with straight line equation:
        • \(y = mx\)

        • \(m = \dfrac{nR}{p}\)

        • \(m \propto \dfrac{1}{p}\)

        • \(p_1 < p_2 < p_3\)

        • \(m_1 > m_2 > m_3\)
