  1. \(\sin x = \dfrac{1}{\csc x} \text{ or, } \csc x = \dfrac{1}{\sin x}\)
  2. \(\cos x = \dfrac{1}{\sec x} \text{ or, } \sec x = \dfrac{1}{\cos x}\)
  3. \(\cot x = \dfrac{1}{\tan x} \text{ or, } \tan x = \dfrac{1}{\cot x}\)
  4. \(\tan x = \dfrac{\sin x}{\cos x} \text{ or, } \cot x = \dfrac{\cos x}{\sin x}\)
  5. \(\sin^2 x + \cos^2 x = 1\)
  6. \(1 + \tan^2 x = \sec^2 x \)
  7. \(1 + \cot^2 x = \csc^2 x \)
  8. \(\sec x - \tan x = \dfrac{1}{\sec x + \tan x}\)
  9. \(\csc x - \cot x = \dfrac{1}{\csc x + \cot x}\)
