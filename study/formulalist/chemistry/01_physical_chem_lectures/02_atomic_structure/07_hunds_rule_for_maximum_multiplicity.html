  1. Orbital filling order and subshell capacities (00:30)
    (Back to topic index)
    • Common filling sequence (up to the level needed here):
      \(1s \rightarrow 2s \rightarrow 2p \rightarrow 3s \rightarrow 3p \rightarrow 4s \rightarrow 3d \rightarrow 4p \rightarrow 4d \rightarrow 4f\)
    • Subshell capacity from number of orbitals:
      • \(s\): 1 orbital \(\Rightarrow\) maximum \(2\) electrons
      • \(p\): 3 orbitals \(\Rightarrow\) maximum \(6\) electrons
      • \(d\): 5 orbitals \(\Rightarrow\) maximum \(10\) electrons
    • Key idea: within the same subshell (e.g., \(2p\)), the orbitals have equal energy (degenerate). This creates multiple possible placements for electrons, so a separate rule is needed for how pairing happens.
  2. Electronic configuration: H, He, Li and start of 2p (02:16)
    (Back to topic index)
    • Hydrogen (\(Z=1\)):
      \(1s^1\)
    • Helium (\(Z=2\)):
      \(1s^2\)
    • Lithium (\(Z=3\)):
      \(1s^2\,2s^1\)
    • Beryllium (\(Z=4\)):
      \(1s^2\,2s^2\)
    • Boron (\(Z=5\)):
      \(1s^2\,2s^2\,2p^1\)
      (The single electron can be placed in any one of the three degenerate \(2p\) orbitals as they all have the same energy.)
    • Carbon (\(Z=6\)):
      \(1s^2\,2s^2\,2p^2\)
      This is where confusion starts as we don't know where to place the 6th electron
  3. Why Hund’s rule is needed in p-orbitals (05:06)
    (Back to topic index)
    • For carbon (\(Z=6\)):
      \(1s^2\,2s^2\,2p^2\)
    • Confusion point: in \(2p^2\), should both electrons go into the same \(2p\) orbital (paired), or should they occupy two different \(2p\) orbitals (unpaired)?
    • Since \(2p_x, 2p_y, 2p_z\) are degenerate (equal energy), energy ordering alone does not decide where the second electron goes. This is exactly the pairing question answered by Hund’s rule.
  4. Hund’s rule: statement and definition (05:56)
    (Back to topic index)
    • Core statement (maximum multiplicity idea): electrons prefer to remain unpaired as far as possible within a subshell.
    • Standard definition used for orbital filling:
      Electrons in a subshell are paired only when all the orbitals of that subshell are half-filled with parallel spins.
    • Practical filling rule for a set of degenerate orbitals (e.g., \(p\), \(d\)):
      • First place one electron in each orbital (single occupancy).
      • Keep spins parallel during this single filling.
      • Only after all orbitals have one electron, start pairing (with opposite spin in the same orbital).
    • Illustration for \(2p\) electrons (sequence idea): \(2p^1 \rightarrow 2p^2 \rightarrow 2p^3\) fill as singles first; pairing starts at \(2p^4\), then \(2p^5\), then \(2p^6\).
  5. Applying Hund’s rule: N and O orbital diagrams (07:13)
    (Back to topic index)
    • Nitrogen (\(Z=7\)) configuration:
      \(1s^2\,2s^2\,2p^3\)
    • Orbital occupancy for \(2p^3\): one electron in each of the three \(2p\) orbitals with parallel spins (all unpaired).
    • Oxygen (\(Z=8\)) configuration:
      \(1s^2\,2s^2\,2p^4\)
    • Orbital occupancy for \(2p^4\): first make \(2p\) half-filled (\(2p^3\)) with parallel spins, then the 4th electron pairs in one orbital (opposite spin).
    • Naming of \(p_x, p_y, p_z\) is a labeling choice; any equivalent distribution consistent with Hund’s rule is acceptable because the three \(p\) orbitals are degenerate.
  6. Generalization to d-subshell and wrap-up (09:10)
    (Back to topic index)
    • For \(d\) subshell: 5 degenerate orbitals \(\Rightarrow\) apply the same rule.
    • Filling pattern idea: place one electron in each of the five \(d\) orbitals first (parallel spins), then start pairing after all five have one electron.
    • Summary: Hund’s rule governs pairing within a subshell; pairing begins only after half-filling (single occupancy) of all degenerate orbitals.
