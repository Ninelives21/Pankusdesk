  1. Molecularity: definition & collision view (01:18)
    (Back to topic index)
    • Molecularity = number of reactant molecules/atoms that participate in an elementary (single-step) reaction.
    • Collision-theory interpretation: number of reactant species that must collide simultaneously and effectively (proper orientation + sufficient energy) to form products.
    • If simultaneously 2 collide \(\Rightarrow\) molecularity \(=2\); if 3 collide \(\Rightarrow\) molecularity \(=3\).
  2. Elementary vs complex: where molecularity is defined (06:40)
    (Back to topic index)
    • Molecularity is defined only for elementary (single-step) reactions.
    • For complex (multi-step) reactions, overall molecularity is not defined.
    • For a multi-step mechanism, molecularity can be defined for individual elementary steps, not for the overall reaction.
  3. Elementary reactions: molecularity, order, naming (08:32)
    (Back to topic index)
    • For an elementary reaction, the experimentally obtained rate law matches stoichiometric coefficients as powers.
      Example: if \(A + B \to C\) is elementary, then rate \(\propto [A]^1[B]^1\).
    • Molecularity of an elementary reaction = number of reactant molecules in the balanced elementary equation.
    • In many elementary reactions, order (sum of powers in rate law) equals molecularity.
    • NCERT example (given elementary):
      \( \mathrm{NH_4NO_2 \to N_2 + 2H_2O} \)
      Molecularity \(=1\) (unimolecular); order \(=1\) if rate \(\propto [\mathrm{NH_4NO_2}]^1\).
    • Example (given elementary):
      \( 2\,\mathrm{HI \to H_2 + I_2} \)
      Rate \(\propto [\mathrm{HI}]^2\); molecularity \(=2\) (bimolecular); order \(=2\).
    • Example (given elementary):
      \( 2\,\mathrm{NO} + \mathrm{O_2 \to 2\,NO_2} \)
      Rate \(\propto [\mathrm{NO}]^2[\mathrm{O_2}]^1\); molecularity \(=3\) (termolecular); order \(=3\).
    • Naming by molecularity:
      • 1 \(\Rightarrow\) unimolecular
      • 2 \(\Rightarrow\) bimolecular
      • 3 \(\Rightarrow\) termolecular
  4. Pseudo-order: solvent/excess reactant cases (18:43)
    (Back to topic index)
    • If one reactant is present in large excess (often solvent), its concentration remains approximately constant during the reaction.
    • Example: hydrolysis of an ester (conceptual form):
      \(\mathrm{CH_3COOC_2H_5 + H_2O \to products}\)
      If \(\mathrm{H_2O}\) is solvent (in excess), then \([\mathrm{H_2O}]\approx \text{constant}\).
      Rate \(\propto [\mathrm{CH_3COOC_2H_5}]^1[\mathrm{H_2O}]^1 \approx k'[\mathrm{CH_3COOC_2H_5}]^1\).
    • Hence the reaction behaves as pseudo-first order:
      • Molecularity (collision requirement) \(\approx 2\) (both species needed)
      • Observed order \(=1\) (because one concentration is constant)
    • Another common pseudo-first order example:
      Inversion of cane sugar:
      \(\mathrm{C_{12}H_{22}O_{11} + H_2O \to C_6H_{12}O_6 (glucose) + C_6H_{12}O_6 (fructose)}\)
      With water as solvent, rate \(\approx k'[\mathrm{C_{12}H_{22}O_{11}}]^1\) \(\Rightarrow\) pseudo-first order.
    • General pseudo-order pattern (when one reactant is in excess):
      • Expected order from elementary stoichiometry: \(a+b+\cdots\)
      • Observed order decreases by 1 for each reactant held effectively constant.
    • Example statement (B in excess):
      If \(2A + B \to C\) is treated with \([B]\) constant, then rate \(\approx k'[A]^2\).
      Observed order \(=2\), while molecularity (for an elementary collision picture) would be \(3\) \(\Rightarrow\) pseudo-second order.
  5. Complex reactions: order fractional; molecularity not defined (33:33)
    (Back to topic index)
    • For a complex reaction (multi-step), the overall rate law does not need to match the overall balanced equation.
    • Example (given complex):
      \(\mathrm{NO_2 + CO \to NO + CO_2}\)
      If experimentally rate \(\propto [\mathrm{NO_2}]^2\), then order \(=2\) even though the balanced equation shows 1 \(\mathrm{NO_2}\).
    • Example mention: if experimentally rate \(\propto [\mathrm{CH_3CHO}]^{3/2}\), then order \(=3/2\) (fractional).
    • Molecularity of the overall complex reaction is not defined; only elementary steps of its mechanism can have molecularity.
  6. Why molecularity \(\le 3\): probability argument (37:04)
    (Back to topic index)
    • Molecularity involves simultaneous and effective collisions.
    • Probability of a simultaneous and properly oriented collision of three molecules is already very low; for four or more it becomes extremely unlikely.
    • Hence in standard treatment: molecularity is taken as \(\le 3\); higher molecularity reactions are rarely observed.
    • Even if an overall equation like \(\mathrm{N_2 + 3H_2 \to 2NH_3}\) is written, molecularity is not taken as 4 (overall molecularity is not assigned this way).
  7. Order vs molecularity: key differences (40:01)
    (Back to topic index)
    • Order is an experimentally determined quantity (from rate law measurements).
    • Molecularity is a theoretical/mechanistic count for an elementary step (count molecules participating in that step).
    • Order is defined for elementary and complex reactions; molecularity is defined only for elementary reactions.
    • Order can be zero, positive, negative, or fractional; molecularity cannot be zero/negative/fractional.
      • Molecularity is a positive integer (commonly 1, 2, or 3).
    • Order can be greater than 3; molecularity is taken as not greater than 3.
    • Fractional stoichiometric coefficients do not imply fractional molecularity.
      Example: if written \(A + \dfrac{B}{2} \to C\), multiply by 2:
      \(2A + B \to 2C\)
      Molecularity (for an elementary collision picture) would be 3, not \(3/2\).
