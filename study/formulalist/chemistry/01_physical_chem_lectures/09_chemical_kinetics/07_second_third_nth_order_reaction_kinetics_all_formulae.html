  2. Second-order: definition, integrated rate law (03:23)
    (Back to topic index)
    • Consider a single-reactant reaction:
      \(A \rightarrow\) products
    • Second order (in \(A\)):
      \(\text{Rate} = k[A]^2\)
    • Rate in differential form (stoichiometric coefficient of \(A\) is 1):
      \(\text{Rate} = -\dfrac{d[A]}{dt}\)
    • Equate and separate variables:
      \(-\dfrac{d[A]}{dt} = k[A]^2 \Rightarrow \dfrac{d[A]}{[A]^2} = -k\,dt\)
    • Integrate using limits: at \(t=0\), \([A]=[A]_0\); at time \(t\), \([A]=[A]_t\):
      \(\displaystyle \int_{[A]_0}^{[A]_t}\dfrac{d[A]}{[A]^2} = -k\int_{0}^{t}dt\)
    • Result (integrated rate law, second order):
      \(\dfrac{1}{[A]_t} - \dfrac{1}{[A]_0} = kt\)
    • Time form (useful for numerical questions):
      \(t = \dfrac{1}{k}\left(\dfrac{1}{[A]_t} - \dfrac{1}{[A]_0}\right)\)
    • Rate vs concentration shape (qualitative):
      \(\text{Rate} \propto [A]^2\) (curved, like \(y=x^2\)).
  3. Second-order: linear plot & slope/intercept (09:14)
    (Back to topic index)
    • Rearranged linear form:
      \(\dfrac{1}{[A]_t} = kt + \dfrac{1}{[A]_0}\)
    • Linear plot for second order:
      • \(y\)-axis: \(\dfrac{1}{[A]_t}\)
      • \(x\)-axis: \(t\)
    • From \(y=mx+c\):
      • Slope \(m = k\) (positive slope).
      • \(y\)-intercept \(c = \dfrac{1}{[A]_0}\).
    • Contrast with common linear plots:
      • Zero order: \([A]\) vs \(t\) is a straight line with negative slope.
      • First order: \(\log [A]\) vs \(t\) is a straight line with negative slope.
      • Second order: \(\dfrac{1}{[A]}\) vs \(t\) is a straight line with positive slope.
  4. Second-order: half-life (12:16)
    (Back to topic index)
    • Half-life definition: at \(t=t_{1/2}\), \([A]_t = \dfrac{[A]_0}{2}\).
    • Substitute into second-order time form:
      \(t_{1/2} = \dfrac{1}{k}\left(\dfrac{1}{[A]_0/2} - \dfrac{1}{[A]_0}\right)\)
    • Simplify:
      \(t_{1/2} = \dfrac{1}{k[A]_0}\)
    • Key dependence:
      \(t_{1/2} \propto \dfrac{1}{[A]_0}\) for second order.
  5. Third-order: integrated rate law (15:03)
    (Back to topic index)
    • For a single-reactant third-order case:
      \(A \rightarrow\) products, \(\text{Rate} = k[A]^3\)
    • Differential form:
      \(-\dfrac{d[A]}{dt} = k[A]^3 \Rightarrow \dfrac{d[A]}{[A]^3} = -k\,dt\)
    • Integrate with limits \([A]_0 \to [A]_t\), \(0 \to t\):
      \(\displaystyle \int_{[A]_0}^{[A]_t}[A]^{-3}\,d[A] = -k\int_0^t dt\)
    • Using \(\int x^{-3}\,dx = -\dfrac{1}{2x^2}\), the result becomes:
      \(\dfrac{1}{2}\left(\dfrac{1}{[A]_t^2} - \dfrac{1}{[A]_0^2}\right) = kt\)
    • Equivalent time form:
      \(t = \dfrac{1}{2k}\left(\dfrac{1}{[A]_t^2} - \dfrac{1}{[A]_0^2}\right)\)
  6. Third-order: linear plot & slope/intercept (20:44)
    (Back to topic index)
    • Rearranged linear form:
      \(\dfrac{1}{[A]_t^2} = 2kt + \dfrac{1}{[A]_0^2}\)
    • Linear plot for third order:
      • \(y\)-axis: \(\dfrac{1}{[A]_t^2}\)
      • \(x\)-axis: \(t\)
    • From \(y=mx+c\):
      • Slope \(m = 2k\) (positive slope).
      • \(y\)-intercept \(c = \dfrac{1}{[A]_0^2}\).
    • Pattern emerging for orders \(\ge 2\): linear plots involve reciprocal powers of concentration.
  7. General \(n\)-th order: linear plot & integrated form (30:30)
    (Back to topic index)
    • For a single-reactant \(n\)-th order reaction (\(n \ge 2\)):
      \(\text{Rate} = k[A]^n\), and \(-\dfrac{d[A]}{dt} = k[A]^n\)
    • Separation and integration gives the standard integrated form:
      \(t = \dfrac{1}{(n-1)k}\left(\dfrac{1}{[A]_t^{\,n-1}} - \dfrac{1}{[A]_0^{\,n-1}}\right)\)
    • Linear plot form:
      \(\dfrac{1}{[A]_t^{\,n-1}} = (n-1)kt + \dfrac{1}{[A]_0^{\,n-1}}\)
    • Graph for \(n\)-th order (\(n \ge 2\)):
      • \(y\)-axis: \(\dfrac{1}{[A]_t^{\,n-1}}\)
      • \(x\)-axis: \(t\)
      • Slope \(= (n-1)k\)
      • Intercept \(= \dfrac{1}{[A]_0^{\,n-1}}\)
    • Examples of the pattern:
      • Second order: plot \(\dfrac{1}{[A]}\) vs \(t\), slope \(k\).
      • Third order: plot \(\dfrac{1}{[A]^2}\) vs \(t\), slope \(2k\).
      • Fourth order: plot \(\dfrac{1}{[A]^3}\) vs \(t\), slope \(3k\).
  8. General \(n\)-th order: half-life formula & proportionality (32:27)
    (Back to topic index)
    • Half-life condition: \([A]_t = \dfrac{[A]_0}{2}\).
    • Substituting into the \(n\)-th order integrated form (\(n \ge 2\)) yields:
      \(t_{1/2} = \dfrac{2^{\,n-1}-1}{(n-1)k[A]_0^{\,n-1}}\)
    • Key proportionality for any order \(n\) (including \(n=0\) and \(n=1\)):
      \(t_{1/2} \propto [A]_0^{\,1-n}\)
    • Practical use: when two half-lives are given at two different initial concentrations, use proportionality instead of memorizing full formulas.
  9. Half-life scaling check for \(n=0,1,2,3\) (36:36)
    (Back to topic index)
    • Recall standard half-lives:
      • Zero order:
        \(t_{1/2} = \dfrac{[A]_0}{2k}\)
      • First order:
        \(t_{1/2} = \dfrac{0.693}{k}\)
      • Second order:
        \(t_{1/2} = \dfrac{1}{k[A]_0}\)
      • Third order (single reactant):
        \(t_{1/2} = \dfrac{3}{2k[A]_0^2}\)
    • Verify proportionality \(t_{1/2} \propto [A]_0^{\,1-n}\):
      • \(n=0\): \(t_{1/2} \propto [A]_0^{1}\) (depends directly on \([A]_0\)).
      • \(n=1\): \(t_{1/2} \propto [A]_0^{0}\) (independent of \([A]_0\)).
      • \(n=2\): \(t_{1/2} \propto [A]_0^{-1}\) (inverse dependence).
      • \(n=3\): \(t_{1/2} \propto [A]_0^{-2}\) (inverse square dependence).
