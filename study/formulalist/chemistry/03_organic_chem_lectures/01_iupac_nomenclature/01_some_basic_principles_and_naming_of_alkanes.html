Some Basic Principles and Naming Of Alkanes

  1. IUPAC Basic rules
    • IUPAC = International Union of Pure and Applied Chemistry.
    • Three things in naming of organic compounds
      1. Prefix :
        1. Primary : \(\text{1}^o\) = rarely used = example: cycle/bicyclo/spiro
        2. Secondary : \(\text{2}^o\) = branches / Substituents (ex: alkyle groups)
      2. Word root = number of Carbons in the main chain = meth,eth etc
      3. Suffix :
        1. Primary : \(\text{1}^o\) = type of C-C bond: ane/ene/yne
        2. Secondary : \(\text{2}^o\) = functional group = example, alcohol, aldehyde etc.
    • Naming = secondary prefix + primary prefix + word root + primary suffix + secondary suffis

  2. Secondary prefix
    • Secondary prefix = branches or substituents (not part of main chain)
    • Their position is very important.
    • Example Halogens (X) : Fluoro, Chloro, Bromo, Iodo
    • -N=O- is "Nitroso"
    • \(\text{N}_3\) : "Azido/Azo"
    • Example -R- = Alkyle groups : methyl (\(\text{CH}_3\)), ethyl (\(\text{C}_2{H}_5\)), propyl (\(\text{C}_3{H}_7\)), phenyl (\(\text{C}_6{H}_5\))
    • -OR- : "Alkoxy"; example: methoxy (\(\text{OCH}_3\)), ethoxy(OEt) (\(\text{OC}_2{H}_5\)), OPh - Phenoxy

  4. Primary Suffix
    • Primary Suffix defines what is the bond between C-C in the main chain
      1. Single bond = "ane"
      2. Double bond = "ene"
      3. Triple bond = "yne"


  5. Nomenclature Rules
    1. Select the longest chain; and this chain need not be straight. This is the main/principal chain.
    2. If two or more longest C-C chains are there, select the one with the maximum number of branches.
    3. Numbering of carbon atoms : numbering should be done in a way that the branch gets the lowest "locant" (location). Also known as the "lowest locant rule".
    4. Note: "bond line notation" = zig zag line instead of straight bond lines. The pointy parts of the zig zag line indicates presence of C-atom.
    5. Numbering as per the "lowest sum rule" : if more than one numbering is happning when there are multiple substituents, add the position of all the substituents and compare the sums. The numbering with the lowest sum wins.
    6. Higher priority than the above = "first point of difference" - start numbering so that you can see a substituent gets the lowest numbering first. This rule dominates over the "lowest sum rule."
    7. Name the substituents alphabetically. Exception is when "di", "tri" etc are there. These are not considered alphabetically.
    8. If there is a tie in first point of difference, then go alphabetically on substituents (example 4 below)


  6. Nomenclature Examples (single bond only)
    1. Example 01 : 5-ethyl-3-methyl-octane \(\rightarrow\) First point vs lowest sum.
    2. Example 02 : 5-ethyl-3,3-dimethyl heptane numberings. \(\rightarrow\) When first point of difference is same for two different numbering + Exception to alphabetical rule.
    3. Example 03 :4-bromo-1-chloro pentane \(\rightarrow\) Alphabetical substituents
    4. Example 04 :1-bromo-4-chloro butane \(\rightarrow\) Tie in first point of difference, then go alphabetically on substituents.
